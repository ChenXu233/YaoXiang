# 第二章：数据的形状

> 本章目标：理解什么是类型，为什么要给数据"贴标签"


## 2.1 计算机里的数据

计算机里存放着各种各样的**数据**：

| 数据 | 例子 |
|------|------|
| 数字 | `42`、`3.14`、`-7` |
| 文字 | `"你好"`、`"Hello"` |
| 是/否 | `true`、`false` |
| 颜色 | 红色、蓝色 |

程序就是处理这些数据的。

## 2.2 什么是类型？

**类型**，就是数据的"种类标签"。

想象你去超市：
- 水果放在**水果区**
- 蔬菜放在**蔬菜区**
- 零食放在**零食区**

类型就像超市的**分区标签**，告诉计算机"这个数据是什么种类"。

```yaoxiang
# 不同类型的数据
age: Int = 25              # 整数类型
price: Float = 19.99       # 小数类型
name: String = "小明"      # 文字类型
is_student: Bool = true    # 是/否类型
```


## 2.3 为什么要用类型？

**问题**：如果没有类型会怎样？

想象超市没有分区：
- 顾客想买苹果，却在一堆袜子中间找
- 店员不知道把新进的货物放在哪里

**类型的作用**：

| 作用 | 说明 | 例子 |
|------|------|------|
| **组织** | 把同类数据放在一起 | 所有 Int 在一起 |
| **限制** | 防止错误的操作 | 不能给文字"加法" |
| **沟通** | 让代码更容易理解 | 看类型就知道用途 |
| **优化** | 计算机可以高效存储 | Int 占 8 字节 |


## 2.4 YaoXiang 的基本类型

YaoXiang 为我们准备了各种基本类型：

| 类型 | 名称 | 例子 | 用途 |
|------|------|------|------|
| `Int` | 整数 | `42`、`-7`、`0` | 计数、编号 |
| `Float` | 小数 | `3.14`、`-0.5` | 金钱、科学计算 |
| `Bool` | 布尔 | `true`、`false` | 是/否判断 |
| `String` | 字符串 | `"你好"`、`"Hi"` | 文字 |
| `Char` | 字符 | `'A'`、`'中'` | 单个文字 |
| `Void` | 空 | `null` | 表示"没有值" |


## 2.5 类型的基本操作

每种类型都有自己可以做的"操作"：

```yaoxiang
# 整数操作
a: Int = 10
b: Int = 3
c: Int = a + b      # 加法：13
d: Int = a * b      # 乘法：30

# 小数操作
x: Float = 3.14
y: Float = x * 2    # 乘法：6.28

# 文字操作
greeting: String = "你好"
name: String = "小明"
message: String = greeting + " " + name  # 拼接："你好 小明"

# 是/否操作
is_adult: Bool = age >= 18
is_weekend: Bool = day == "Saturday"
```


## 2.6 类型推导

写代码时，有时候类型可以**自动推导**：

```yaoxiang
# 完整写法（推荐初学者使用）
age: Int = 18

# 简短写法（编译器自动推导类型）
age = 18           # 推导为 Int
price = 19.99      # 推导为 Float
name = "小明"      # 推导为 String
is_done = true     # 推导为 Bool
```

**建议**：初学者最好**显式写出类型**，这样代码更容易理解！

## 2.7 类型的转换

不同类型之间可以转换：

```yaoxiang
# 整数转小数
num: Int = 42
num_float: Float = num.as_float()  # 42.0

# 小数转整数（会舍去小数部分）
pi: Float = 3.14
pi_int: Int = pi.as_int()          # 3

# 数字转文字
age: Int = 25
age_str: String = age.as_string()  # "25"
```


## 2.8 本章小结

| 概念 | 理解 |
|------|------|
| 数据 | 程序处理的信息（数字、文字、是/否等） |
| 类型 | 数据的"种类标签"，告诉计算机数据是什么 |
| 类型的作用 | 组织、限制、沟通、优化 |
| 基本类型 | Int、Float、Bool、String、Char、Void |

> 「天地之大德曰生，圣人之大宝曰位。」
> —— 《周易·系辞上传》
>
> 天地生万物，各有其性、各归其位。
