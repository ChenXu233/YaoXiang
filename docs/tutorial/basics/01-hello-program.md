# 第一章：Hello，程序

> 本章目标：理解什么是程序，代码是如何变成运行的程序的


## 1.1 从一个问题开始

你有没有想过，手机里的APP、网页上的按钮、还有你玩的游戏，它们是怎么工作的？

答案是：**它们都是程序**。

程序，就是一系列给计算机的**指令**。就像食谱告诉厨师怎么做菜，程序告诉计算机怎么做事情。


## 1.2 什么是代码？

**代码**，就是用编程语言写的程序。

我们平常说话用的是中文、英文，程序员和计算机"说话"用的是编程语言。YaoXiang 就是一种编程语言。

```yaoxiang
# 这是一行 YaoXiang 代码
print("Hello, 世界")
```

这行代码的意思是：**在屏幕上显示"Hello, 世界"**。

> **小知识**：代码里 `#` 后面的文字是"注释"，计算机不会执行它们，只是给人看的说明。


## 1.3 从代码到程序

人和计算机使用的"语言"不同，所以我们写的代码需要经过一个"翻译"过程：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   源代码    │ ──▶ │   编译器    │ ──▶ │   可执行程序 │
│  (YaoXiang)│     │   翻译官    │     │  (计算机能看懂)│
└─────────────┘     └─────────────┘     └─────────────┘
```

这个"翻译官"叫做**编译器**。YaoXiang 的编译器叫做 `yaoxiang`。

## 1.4 第一个程序

让我们写第一个 YaoXiang 程序！

```yaoxiang
# hello.yx
main: () -> Void = {
    print("Hello, 我是你的第一个程序！")
}
```

**逐行解释**：

| 代码 | 意思 |
|------|------|
| `main:` | 这是一个叫"main"的函数，程序从这里开始执行 |
| `()` | 这个函数不需要输入参数 |
| `-> Void` | 这个函数不返回任何值 |
| `{ ... }` | 大括号里的内容是函数要执行的指令 |
| `print("...")` | 在屏幕上显示引号里的文字 |

> **注意**：YaoXiang 代码必须使用 **4个空格** 缩进，不能用 Tab！

## 1.5 运行程序

把上面的代码保存为 `hello.yx`，然后在终端运行：

```bash
yaoxiang hello.yx
```

你会看到屏幕上显示：

```
Hello, 我是你的第一个程序！
```

**恭喜！你已经写出并运行了第一个程序！** 🎉


## 1.6 程序的结构

让我们再看一下程序的完整结构：

```yaoxiang
# 文件名：hello.yx
main: () -> Void = {
    # 在这里写指令
    print("Hello, 世界")
}
```

- **函数**：一个独立的功能单元
- **main**：程序入口，所有程序都从这里开始
- **print**：YaoXiang 内置的"打印"功能

## 1.7 本章小结

| 概念 | 理解 |
|------|------|
| 程序 | 给计算机的一系列指令 |
| 代码 | 用编程语言写的程序 |
| 编译器 | 把代码翻译成计算机能看懂的语言 |
| 函数 | 程序的基本功能单元 |
| main | 程序的入口点 |

## 1.8 易经引言
