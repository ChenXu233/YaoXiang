# Добро пожаловать в мир YaoXiang!

> «Бесконечность порождает Великий Предел. Великий Предел при движении порождает Ян, при покое — Инь.»
> — Чжоу Дунь-и «Рассуждение о Великом Пределе»

Здесь находится центр мира YaoXiang.

В восточной философии «яо» — это базовые символы, образующие гексаграммы, где заключены все изменения инь и ян. «Сян» — это внешнее проявление сущности вещей, у всего в мире есть закономерности.

**Язык программирования YaoXiang** — это попытка интегрировать эту древнюю мудрость в современную систему типов.

## Что такое YaoXiang?

Представьте, что вы собираете конструктор LEGO:

| Концепция | LEGO | YaoXiang |
|-----------|------|----------|
| Базовые единицы | Блоки | **Типы** |
| Способ комбинирования | Сборка | **Функции** |
| Результат | Самолёты, машины | **Программы** |

В мире YaoXiang **всё является типом**:
- Числа — это типы, текст — это типы
- Типы сами являются типами (мета-типы)
- Функции — это мосты от типа к типу

## Основные возможности

| Возможность | YaoXiang | Примеры из других языков |
|-------------|----------|--------------------------|
| Type system | Зависимые типы + параметрический полиморфизм | TypeScript |
| Управление памятью | Модель владения (без GC) | Rust |
| Асинхронное программирование | Модель spawn (незаметный параллелизм) | async/await |
| Стиль синтаксиса | В стиле Python | Python |

**Одной фразой**: так же плавно, как Python, так же безопасно, как Rust.
