# Добро пожаловать во вселенную YaoXiang!

> «Беспредельное порождает Великий Предел. Великий Предел при движении рождает Ян, при покое — Инь.»
> —— Чжоу Дунь-и «Рассуждение о Тайцзи»

Здесь находится центр вселенной YaoXiang.

В восточной философии «яо» — это основной символ, составляющий гексаграмму, в котором заключены все изменения инь и ян. «Сян» — это внешнее проявление сущности вещей, у всего в мире есть свои закономерности.

**Язык программирования YaoXiang** — это попытка интегрировать эту древнюю мудрость в современную систему типов.

## Что такое YaoXiang?

Представьте, что вы собираете конструктор LEGO:

| Концепция | LEGO | YaoXiang |
|-----------|------|----------|
| Базовая единица | Блок | **Тип** |
| Способ комбинирования | Соединение | **Функциональное отображение** |
| Результат | Самолёт, машина | **Программа** |

В мире YaoXiang **всё является типом**:
- Числа — это типы, текст — это типы
- Типы сами по себе тоже являются типами (метатипы)
- Функции — это мосты от типа к типу

## Ключевые особенности

| Особенность | YaoXiang | Примеры в других языках |
|-------------|----------|------------------------|
| Система типов | Зависимые типы + параметрический полиморфизм | TypeScript |
| Управление памятью | Модель владения (без GC) | Rust |
| Асинхронное программирование | Модель spawn (бесшовный параллелизм) | async/await |
| Стиль синтаксиса | Стиль Python | Python |

**Одними словами**: так же плавно, как Python, так же безопасно, как Rust.
