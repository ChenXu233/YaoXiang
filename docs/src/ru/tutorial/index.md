# Добро пожаловать во вселенную YaoXiang!

> «Бесконечность порождает великий предел. Великий предел приходит в движение и рождает Ян, пребывает в покое и рождает Инь.»
> — Чжоу Дуньи «Рассуждение о великом пределе»

Здесь находится центр вселенной YaoXiang.

В восточной философии «яо» — это основной символ, составляющий гексаграммы, в котором заключены все изменения инь и ян. «Сян» — это внешнее проявление сущности вещей, у всего в мире есть свои закономерности.

**Язык программирования YaoXiang** — это попытка интегрировать эту древнюю мудрость в современную систему типов.

## Что такое YaoXiang?

Представьте, что вы строите из кубиков LEGO:

| Концепция | LEGO | YaoXiang |
|-----------|------|----------|
| Базовая единица | Кубик | **Тип** |
| Способ комбинирования | Соединение | **Функциональное отображение** |
| Результат | Самолёт, машина | **Программа** |

В мире YaoXiang **всё является типом**:
- Числа — это типы, текст — это типы
- Типы сами по себе тоже являются типами (метатипы)
- Функции — это мосты от типа к типу

## Ключевые особенности

| Особенность | YaoXiang | Примеры из других языков |
|-------------|----------|--------------------------|
| Система типов | Зависимые типы + параметрический полиморфизм | TypeScript |
| Управление памятью | Модель владения (без GC) | Rust |
| Асинхронное программирование | Модель спавна (незаметный параллелизм) | async/await |
| Синтаксический стиль | Стиль Python | Python |

**Одними словами**: так же плавно, как на Python, так же безопасно, как в Rust.
