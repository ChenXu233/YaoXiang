# Добро пожаловать во вселенную YaoXiang!

> «Беспредельное рождает Великий Предел. Движение Великого Предела рождает Ян, покой порождает Инь.»
> — Чжоу Дуньи, «Тайцзи ту шо»

Здесь находится центр вселенной YaoXiang.

В восточной философии «Яо» — это базовые символы, из которых складываются гексаграммы; в них заключены все превращения Инь и Ян. «Сян» — это внешнее проявление сущности вещей; во всех явлениях мира можно усмотреть закономерности.

**Язык программирования YaoXiang** — это попытка перенести эту древнюю мудрость в современную систему типов.

## Что такое YaoXiang?

Представьте, что вы собираете конструктор Lego:

| Понятие | Lego | YaoXiang |
|------|------|------|
| Базовая единица | Кирпичик | **Тип** |
| Способ сочетания | Соединение | **Отображение функций** |
| Готовое изделие | Самолёт, автомобиль | **Программа** |

В мире YaoXiang **всё является типом**:
- Числа — это тип, текст — это тип
- Сами типы — это тоже тип (мета-тип)
- Функции — это мост от типа к типу

## Ключевые особенности

| Особенность | YaoXiang | Примеры в других языках |
|------|------|-------------|
| Система типов | Зависимые типы + параметрический полиморфизм | TypeScript |
| Управление памятью | Модель ownership (без GC) | Rust |
| Асинхронное программирование | spawn-модель (неявная конкурентность) | async/await |
| Стиль синтаксиса | В духе Python | Python |

**Одной фразой**: плавно, как Python; безопасно, как Rust.
