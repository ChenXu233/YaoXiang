# Цикл while

`for` подходит для сценариев «обхода известной коллекции», а `while` — для другого случая — **когда вы не знаете, сколько раз нужно повторить цикл, но знаете, когда нужно остановиться**.

## Базовый синтаксис

Определение оператора `while` в спецификации синтаксиса:

```
while Expr Block
```

Структура проста: после `while` следует условное выражение, затем блок кода тела цикла. Пока условие равно `true`, тело цикла продолжает выполняться.

```yaoxiang
mut count = 1

while count <= 5 {
    print(count)
    count = count + 1
}
// Вывод: 1 2 3 4 5
```

Обратите внимание, что мы объявляем переменную через `mut count`, — потому что `count` нужно изменять в цикле. Если написать `count = 1` (неизменяемой), то `count = count + 1` в теле цикла вызовет ошибку.

## Порядок выполнения while

Шаги выполнения цикла `while` следующие:

1. Проверить условное выражение
2. Если условие равно `true`, выполнить тело цикла, затем вернуться к шагу 1
3. Если условие равно `false`, завершить цикл и продолжить выполнение последующего кода

Условие проверяется **перед началом каждой итерации**. Если условие изначально равно `false`, тело цикла не выполнится ни разу:

```yaoxiang
mut n = 0
while n > 0 {
    print("Эта строка никогда не будет напечатана")
    n = n - 1
}
// Условие n > 0 сразу false, тело цикла пропускается
```

## break: досрочный выход из цикла

Иногда нужно досрочно выйти из цикла — например, когда искомый элемент уже найден:

```yaoxiang
numbers = [3, 7, 2, 9, 5]
mut found = false
mut index = 0

while index < 5 {
    if numbers[index] == 9 {
        found = true
        break      // Нашли, продолжать не нужно
    }
    index = index + 1
}

print("Нашли? " + found.to_string())  // "Нашли? true"
```

`break` немедленно выводит программу из текущего цикла и продолжает выполнение кода после цикла.

## continue: пропуск текущей итерации

`continue` отличается от `break` — он не выходит из цикла, а пропускает оставшуюся часть текущей итерации и сразу переходит к следующей проверке условия:

```yaoxiang
mut n = 0
while n < 5 {
    n = n + 1
    if n == 3 {
        continue   // Пропускаем 3, не печатаем
    }
    print(n)
}
// Вывод: 1 2 4 5
```

В этом коде, когда `n` равно 3, `continue` пропускает `println(n)` и сразу возвращается к проверке условия `while n < 5`.

## Избегание бесконечного цикла

При использовании `while` нужно быть особенно внимательным — убедитесь, что условие цикла в конечном итоге станет `false`, иначе программа зависнет навсегда:

```yaoxiang
// Опасно! Бесконечный цикл — условие всегда true
// mut x = 1
// while x > 0 {
//     x = x + 1     // x всё растёт, никогда не станет <= 0
// }

// Правильно — есть явное условие завершения
mut x = 1
while x <= 5 {
    print(x)
    x = x + 1        // x постепенно растёт, в конце концов x > 5, цикл завершится
}
```

## Использование while для чтения ввода

Классический сценарий использования `while` — обработка ввода неизвестной длины: вы не знаете, сколько строк введёт пользователь, но знаете, что «при пустом вводе нужно остановиться»:

```yaoxiang
// Псевдокод — типичное использование while
// read_line возвращает пустую строку при чтении пустой строки
mut line = read_line()
while line != "" {
    process(line)
    line = read_line()
}
```

Этот шаблон «проверить условие → обработать данные → обновить условие» — основной способ использования `while`.

| Сценарий использования | Неизвестно число итераций, известно только условие завершения |

| Порядок выполнения | Сначала проверка условия, затем выполнение тела цикла |
