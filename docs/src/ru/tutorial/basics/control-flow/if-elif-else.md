# if-elif-else

`if-elif-else` — это самый базовый инструмент принятия решений в программировании. Его логика очень интуитивна: **если условие выполняется, выполнить один блок кода; иначе проверить следующее условие; если ни одно не выполняется — перейти к ветке по умолчанию**.

## Базовый синтаксис

В спецификации синтаксиса определения выражений `if` и инструкций `if` полностью совпадают:

```
if Expr Block ('elif' Expr Block)* ('else' Block)?
```

Перевод на повседневный язык: `if`, затем выражение условия и блок кода, после чего может идти от нуля и более пар `elif условие блок`, и в конце — необязательный `else блок`.

Самая простая форма — только `if`:

```yaoxiang
if temperature > 30 {
    print("Жарко, включи кондиционер")
}
```

Добавляем `else`:

```yaoxiang
if is_raining {
    print("Возьми зонт")
} else {
    print("Зонт не нужен")
}
```

Несколько условий через `elif`:

```yaoxiang
score = 85

if score >= 90 {
    print("Отлично")
} elif score >= 80 {
    print("Хорошо")
} elif score >= 60 {
    print("Удовлетворительно")
} else {
    print("Нужно стараться")
}
```

Обратите внимание, что ключевое слово в YaoXiang — `elif`, а не `else if`. Это проявление намеренного стремления языка к минимализму в наборе ключевых слов.

## if — это выражение

Это одна из самых важных особенностей управляющих конструкций YaoXiang: **`if` может использоваться как выражение, возвращающее значение**.

```yaoxiang
// Выражение if: значение из каждой ветки присваивается result
result = if x > 0 {
    "Положительное"
} elif x < 0 {
    "Отрицательное"
} else {
    "Ноль"
}
// Теперь result — одно из "Положительное", "Отрицательное" или "Ноль"
```

Когда `if` используется как выражение, типы возвращаемых значений всех ветвей должны совпадать:

```yaoxiang
score = 88

// Все ветви возвращают String, типы совпадают, всё в порядке
grade = if score >= 90 {
    "A"
} elif score >= 80 {
    "B"
} elif score >= 60 {
    "C"
} else {
    "D"
}
print(grade)  // "B"
```

В блоке кода каждой ветки **значение последнего выражения является возвращаемым значением этой ветки**. Можно также явно вернуть значение через `return`, но в ветках обычно достаточно просто написать выражение.

```yaoxiang
// Просто выражение — рекомендуемый способ
category = if age < 18 { "Несовершеннолетний" } else { "Совершеннолетний" }

// Можно и явно через return — результат тот же
category = if age < 18 {
    return "Несовершеннолетний"
} else {
    return "Совершеннолетний"
}
```

Если вы используете `if` только для проверки условия и вам не нужно значение, это обычная инструкция — она полностью совместима с формой выражения.

## Вложенные if

Внутри `if` можно писать ещё один `if`, чтобы обрабатывать многоуровневые условия:

```yaoxiang
age = 25
has_ticket = true

if age >= 18 {
    if has_ticket {
        print("Добро пожаловать")
    } else {
        print("Сначала купите билет")
    }
} else {
    print("Несовершеннолетним нужно сопровождение взрослых")
}
```

При вложении выражений в YaoXiang нет неоднозначности «висячего else», как в языке C — каждый `else` всегда относится к ближайшему ещё не спаренному `if`.

## Комбинирование условий с булевыми операторами

В условиях можно использовать `and`, `or`, `not` для комбинирования нескольких проверок:

```yaoxiang
username = "admin"
password = "123456"

// and: оба условия должны выполняться
if username == "admin" and password == "123456" {
    print("Вход выполнен")
}

// or: выполняется хотя бы одно условие
if role == "admin" or role == "moderator" {
    print("Есть права администратора")
}

// not: отрицание
if not is_banned {
    print("Можно писать")
}

// Комбинированное использование
if (age >= 18 and age <= 60) or is_vip {
    print("Можно участвовать в мероприятии")
}
```

По приоритету операторов: `not` выше, чем `and`, а `and` выше, чем `or`. Если сомневаетесь — ставьте скобки, чтобы намерение было яснее.

## Итоги

| Пункт | Описание |
|------|------|
| Базовая структура | `if условие { ... } elif условие { ... } else { ... }` |
| elif | В YaoXiang используется `elif`, а не `else if` |
| Выражение | `if` может возвращать значение; типы всех ветвей должны совпадать |
| Возвращаемое значение ветви | Значение последнего выражения в блоке ветки является возвращаемым значением |
| Вложенность | Внутри `if` можно писать ещё один `if`, неоднозначности висячего else нет |
| Булевы операторы | `and`, `or`, `not` для комбинирования условий |
