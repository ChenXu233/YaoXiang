# F-string

f-string — это **шаблонная строка** в YaoXiang: вы можете встраивать переменные и выражения прямо в строку, а компилятор автоматически выполнит преобразование типов и конкатенацию.

## Базовое использование

Добавьте префикс `f` перед строкой и используйте `{выражение}` для вставки значений:

```yaoxiang
name = "Alice"
age = 25

greeting = f"Hello {name}, you are {age} years old"
print(greeting)  // Hello Alice, you are 25 years old
```

Сравните с традиционным способом конкатенации — разница очевидна:

```yaoxiang
// ❌ Традиционная конкатенация: громоздко и легко ошибиться
message = "Hello ".concat(name).concat(", age: ").concat(age.to_string())

// ✅ f-string: наглядно и кратко
message = f"Hello {name}, age: {age}"
```

## Интерполяция выражений

Внутри `{}` можно использовать не только переменные, но и любые выражения:

```yaoxiang
x = 10
y = 20

print(f"Sum: {x + y}")         // Sum: 30
print(f"Product: {x * y}")     // Product: 200
print(f"Is positive? {x > 0}") // Is positive? true
```

## Спецификаторы формата

После выражения добавьте `:` и спецификатор формата, чтобы управлять выводом:

```yaoxiang
pi = 3.14159265

print(f"Pi: {pi}")       // Pi: 3.14159265
print(f"Pi: {pi:.2f}")   // Pi: 3.14 (2 знака после запятой)
print(f"Pi: {pi:.4f}")   // Pi: 3.1416 (4 знака после запятой)
```

Часто используемые спецификаторы формата:

| Спецификатор | Значение | Пример | Вывод |
|--------------|----------|--------|-------|
| `:.2f` | Число с плавающей точкой, 2 знака | `f"{3.14159:.2f}"` | `3.14` |
| `:d` | Десятичное целое | `f"{42:d}"` | `42` |
| `:x` | Шестнадцатеричное | `f"{255:x}"` | `ff` |
| `:e` | Экспоненциальная запись | `f"{1000:e}"` | `1.000000e+03` |
| `:s` | Строка | `f"{name:s}"` | `hello` |

## Вызов методов

Внутри `{}` можно вызывать методы:

```yaoxiang
name = "alice"

print(f"Upper: {name.uppercase()}")   // Upper: ALICE
print(f"Length: {name.len()}")        // Length: 5
```

## Экранирование фигурных скобок

Если нужно вывести литералы `{` или `}`, просто **удвойте** их:

```yaoxiang
print(f"{{literal braces}}")     // {literal braces}
print(f"Set: {{1, 2, 3}}")       // Set: {1, 2, 3}

// Смешанный случай: удвоенные выводят литерал {, одиночные — это интерполяция
name = "YaoXiang"
print(f"{{name}} is {name}")     // {name} is YaoXiang
```

## Многострочные f-string

f-string может занимать несколько строк:

```yaoxiang
name = "Alice"
age = 25
city = "Beijing"

info = f"""
Name: {name}
Age: {age}
City: {city}
"""

print(info)
// Name: Alice
// Age: 25
// City: Beijing
```

## Как работает f-string

Когда компилятор встречает f-string, он преобразует её в эффективную конкатенацию строк:

```yaoxiang
// То, что вы пишете
f"Hello {name}, age: {age}"

// Во что компилятор это превращает
"Hello ".concat(name.to_string()).concat(", age: ").concat(age.to_string())
```

Это означает, что f-string не только проще в написании, но и по производительности во время выполнения не уступает ручной конкатенации — **нулевые накладные расходы**.
