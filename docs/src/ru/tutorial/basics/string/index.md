# F-string

f-string — это **шаблонная строка** (YaoXiang **模板字符串**) в YaoXiang — вы можете встраивать переменные и выражения прямо в строку, компилятор автоматически выполняет преобразование типов и конкатенацию.

## Базовое использование

Добавьте префикс `f` перед строкой и используйте `{выражение}` для вставки значения:

```yaoxiang
name = "Alice"
age = 25

greeting = f"Hello {name}, you are {age} years old"
println(greeting)  # Hello Alice, you are 25 years old
```

Сравните с традиционным способом конкатенации — разница очевидна:

```yaoxiang
# ❌ Традиционная конкатенация: громоздко и подвержено ошибкам
message = "Hello ".concat(name).concat(", age: ").concat(age.to_string())

# ✅ f-string: интуитивно и лаконично
message = f"Hello {name}, age: {age}"
```

## Интерполяция выражений

`{}` не ограничивается переменными — туда можно поместить любое выражение:

```yaoxiang
x = 10
y = 20

println(f"Sum: {x + y}")         # Sum: 30
println(f"Product: {x * y}")     # Product: 200
println(f"Is positive? {x > 0}") # Is positive? true
```

## Спецификаторы форматирования

Добавьте `:` и спецификатор формата после выражения, чтобы управлять форматом вывода:

```yaoxiang
pi = 3.14159265

println(f"Pi: {pi}")       # Pi: 3.14159265
println(f"Pi: {pi:.2f}")   # Pi: 3.14 (2 знака после запятой)
println(f"Pi: {pi:.4f}")   # Pi: 3.1416 (4 знака после запятой)
```

Распространённые спецификаторы форматирования:

| Спецификатор | Значение | Пример | Вывод |
|--------|------|------|------|
| `:.2f` | Число с плавающей точкой, 2 знака | `f"{3.14159:.2f}"` | `3.14` |
| `:d` | Десятичное целое | `f"{42:d}"` | `42` |
| `:x` | Шестнадцатеричное | `f"{255:x}"` | `ff` |
| `:e` | Экспоненциальная запись | `f"{1000:e}"` | `1.000000e+03` |
| `:s` | Строка | `f"{name:s}"` | `hello` |

## Вызов методов

Внутри `{}` можно вызывать методы:

```yaoxiang
name = "alice"

println(f"Upper: {name.uppercase()}")   # Upper: ALICE
println(f"Length: {name.len()}")        # Length: 5
```

## Экранирование фигурных скобок

Если нужно вывести литеральные `{` или `}`, просто **удвойте** их:

```yaoxiang
println(f"{{literal braces}}")     # {literal braces}
println(f"Set: {{1, 2, 3}}")       # Set: {1, 2, 3}

# Смешивание: двойные выводят литеральную {, одинарные — интерполяцию
name = "YaoXiang"
println(f"{{name}} is {name}")     # {name} is YaoXiang
```

## Многострочный f-string

f-string может быть многострочным:

```yaoxiang
name = "Alice"
age = 25
city = "Beijing"

info = f"""
Name: {name}
Age: {age}
City: {city}
"""

println(info)
# Name: Alice
# Age: 25
# City: Beijing
```

## Как работает f-string

Когда компилятор видит f-string, он преобразует его в эффективную конкатенацию строк:

```yaoxiang
# То, что вы пишете
f"Hello {name}, age: {age}"

# Результат преобразования компилятора
"Hello ".concat(name.to_string()).concat(", age: ").concat(age.to_string())
```

Это означает, что f-string не только проще в написании, но и производительность во время выполнения сопоставима с ручной конкатенацией — **без дополнительных накладных расходов** (zero overhead).
