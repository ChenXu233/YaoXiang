# Списки и словари

Структуры данных — это каркас программы. YaoXiang предоставляет два встроенных типа коллекций: списки и словари.

## Список

Список — это **упорядоченная** последовательность значений, все элементы которой имеют один и тот же тип. Для создания используется синтаксис `[]`:

```yaoxiang
// Создание списка
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]
empty: List(Int) = []       // Пустой список требует аннотации типа
```

### Доступ по индексу

Доступ к элементам по позиции осуществляется с помощью `[]`, индексация начинается с 0:

```yaoxiang
scores = [95, 87, 73, 91]

first = scores[0]    // 95
second = scores[1]   // 87
last = scores[3]     // 91
```

### Основные операции

```yaoxiang
mut items = [1, 2, 3]

// Добавление элемента
items.append(4)       // [1, 2, 3, 4]

// Длина
count = items.len()   // 4

// Срез
slice = items[0..2]   // [1, 2]
```

### Генератор списков

Генератор списков — мощный инструмент для создания списков: он позволяет формировать новые списки на основе существующих:

```yaoxiang
// Базовый генератор
squares = [x * x for x in [1, 2, 3, 4, 5]]
print(squares)  // [1, 4, 9, 16, 25]

// Генератор с условием фильтрации
evens = [x for x in [1, 2, 3, 4, 5, 6] if x % 2 == 0]
print(evens)  // [2, 4, 6]

// Преобразование типов
names = ["Alice", "Bob", "Charlie"]
lengths = [n.len() for n in names]
print(lengths)  // [5, 3, 7]
```

Синтаксис: `[выражение for переменная in список if условие]` — часть `if условие` является необязательной.

## Словарь

Словарь — это коллекция **пар ключ-значение**, в которой ключи являются строками, а значения могут иметь произвольный тип. Для создания используется синтаксис `{}`:

```yaoxiang
// Создание словаря
scores = {"Alice": 90, "Bob": 85, "Charlie": 92}
empty: Dict(Int) = {}          // Пустой словарь требует аннотации типа
```

### Доступ по ключу

Доступ к значениям по ключу осуществляется с помощью `[]`:

```yaoxiang
scores = {"Alice": 90, "Bob": 85}

alice = scores["Alice"]   // 90
bob = scores["Bob"]       // 85
```

### Изменение словаря

```yaoxiang
mut data = {"name": "Alice"}

// Добавление/обновление пары ключ-значение
data["age"] = 25
data["name"] = "Bob"

print(data)  // {"name": "Bob", "age": 25}
```

### Проверка наличия ключа

Оператор `in` используется для проверки существования ключа:

```yaoxiang
config = {"host": "localhost", "port": "8080"}

has_host = "host" in config    // true
has_user = "user" in config    // false
```


## Итоги

| Тип | Синтаксис | Упорядочен? | Допускает дублирование? | Тип ключа |
|------|------|--------|----------|--------|
| Список | `[1, 2, 3]` | ✅ | ✅ | целочисленный индекс |
| Словарь | `{"a": 1}` | ✅ | ключи уникальны | String |

Список — ваш основной контейнер, а словарь идеально подходит для поиска по ключу.
