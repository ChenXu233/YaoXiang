# Списки и словари

Структуры данных — это скелет программы. YaoXiang предоставляет три встроенных типа коллекций: список, словарь и множество.

## Списки

Список — это **упорядоченная** последовательность значений одного типа. Создаётся с помощью `[]`:

```yaoxiang
# Создание списка
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]
empty: List(Int) = []       # Для пустого списка требуется аннотация типа
```

### Доступ по индексу

Доступ к элементам по позиции осуществляется с помощью `[]`, индексы начинаются с 0:

```yaoxiang
scores = [95, 87, 73, 91]

first = scores[0]    # 95
second = scores[1]   # 87
last = scores[3]     # 91
```

### Основные операции

```yaoxiang
mut items = [1, 2, 3]

# Добавление элемента
items.append(4)       # [1, 2, 3, 4]

# Длина
count = items.len()   # 4

# Срез
slice = items[0..2]   # [1, 2]
```

### Генераторы списков

Генератор списка — это мощный инструмент для создания списков: новый список формируется на основе уже существующего:

```yaoxiang
# Базовый генератор
squares = [x * x for x in [1, 2, 3, 4, 5]]
println(squares)  # [1, 4, 9, 16, 25]

# Генератор с условием фильтрации
evens = [x for x in [1, 2, 3, 4, 5, 6] if x % 2 == 0]
println(evens)  # [2, 4, 6]

# Преобразование типа
names = ["Alice", "Bob", "Charlie"]
lengths = [n.len() for n in names]
println(lengths)  # [5, 3, 7]
```

Синтаксис: `[выражение for переменная in список if условие]` — часть `if условие` необязательна.

## Словари

Словарь — это набор **пар ключ-значение**, где ключи являются строками, а значения могут быть любого типа. Создаётся с помощью `{}`:

```yaoxiang
# Создание словаря
scores = {"Alice": 90, "Bob": 85, "Charlie": 92}
empty: Dict(Int) = {}          # Для пустого словаря требуется аннотация типа
```

### Доступ по ключу

Получение значения по ключу с помощью `[]`:

```yaoxiang
scores = {"Alice": 90, "Bob": 85}

alice = scores["Alice"]   # 90
bob = scores["Bob"]       # 85
```

### Изменение словаря

```yaoxiang
mut data = {"name": "Alice"}

# Добавление/обновление пары ключ-значение
data["age"] = 25
data["name"] = "Bob"

println(data)  # {"name": "Bob", "age": 25}
```

### Проверка членства

Ключевое слово `in` проверяет наличие ключа:

```yaoxiang
config = {"host": "localhost", "port": "8080"}

has_host = "host" in config    # true
has_user = "user" in config    # false
```

## Множества

Множество — это набор **неупорядоченных, уникальных** значений. Создаётся с помощью `{}` (отличие от словаря: отсутствие двоеточия):

```yaoxiang
# Создание множества
colors = {"red", "green", "blue"}
numbers = {1, 2, 3, 3, 2, 1}   # Дубликаты удаляются автоматически

println(numbers)  # {1, 2, 3}
```

### Операции с множествами

```yaoxiang
mut tags = {"rust", "compiler"}

# Добавление
tags.insert("language")

# Проверка членства
has_rust = "rust" in tags      # true
has_python = "python" in tags  # false
```

## Итоги

| Тип | Синтаксис | Упорядочен? | Допускает дубликаты? | Тип ключа |
|-----|-----------|-------------|----------------------|-----------|
| Список | `[1, 2, 3]` | ✅ | ✅ | Целочисленный индекс |
| Словарь | `{"a": 1}` | ✅ | Ключи уникальны | String |
| Множество | `{1, 2, 3}` | ❌ | ❌ | Нет |

Список — ваш основной контейнер, словарь подходит для поиска по ключу, а множество — для удаления дубликатов и проверки членства.
