## Жизненный цикл и судьба

RFC имеет следующие статусы перехода:

```
┌─────────────┐
│  Черновик   │  ← Создаётся автором
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ На рассмотрении │  ← Обсуждение сообществом
└──────┬──────┘
       │
       ├──────────────────┐
       ▼                  ▼
┌─────────────┐    ┌─────────────┐
│  Принят     │    │ Отклонён    │
└──────┬──────┘    └──────┬──────┘
       │                  │
       ▼                  ▼
┌─────────────┐    ┌─────────────┐
│   accepted/ │    │    rfc/     │
│(официальный │    │(остаётся   │
│   дизайн)   │    │  на месте)  │
└─────────────┘    └─────────────┘
```

### Описание статусов

| Статус | Расположение | Описание |
|------|------|------|
| **Черновик** | `docs/design/rfc/` | Авторский черновик, ожидает отправки на рассмотрение |
| **На рассмотрении** | `docs/design/rfc/` | Открыто для обсуждения и обратной связи сообщества |
| **Принят** | `docs/design/accepted/` | Становится официальным документом дизайна, переходит в стадию реализации |
| **Отклонён** | `docs/design/rfc/` | Остаётся в каталоге RFC, статус обновляется |

### Действия после принятия

1. Переместить RFC в каталог `docs/design/accepted/`
2. Обновить имя файла на описательное (например, `enhanced-pattern-matching.md`)
3. Обновить статус на "Официальный"
4. Обновить статус на "Принят", добавить дату принятия

### Действия после отклонения

1. Оставить в каталоге `docs/design/rfc/`
2. Добавить в начало файла причину и дату отклонения
3. Обновить статус на "Отклонён"

### Действия после завершения обсуждения

Когда по открытому вопросу достигнут консенсус:

1. **Обновить Приложение А**: Заполнить «Решение» в теме обсуждения
2. **Обновить основной текст**: Синхронизировать решение с документом
3. **Записать решение**: Добавить в «Приложение Б: Записи дизайнерских решений»
4. **Отметить вопрос**: Отметить галочкой `[x]` в списке «Открытые вопросы»

---

> **Примечание**: Номер RFC используется только на этапе обсуждения. После принятия номер удаляется, вместо него используется описательное имя файла.
