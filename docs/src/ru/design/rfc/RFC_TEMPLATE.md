#### Правила управления обсуждениями

| Этап | Действие |
|------|------|
| В процессе обсуждения | Записать в приложении, сохранить статус открытого |
| Решено | Обновить решение, отметить статус, **обновить соответствующий контент в основном тексте** |
| Отклонено | Указать причину отклонения, сохранить запись для справки |

> **Важно**: после завершения обсуждения необходимо обновить соответствующий контент в основном тексте! Записи обсуждений можно сохранить для исторической справки.
