### Описание статусов

| Статус | Расположение | Описание |
|--------|--------------|----------|
| **Черновик** | `docs/reference/rfc/draft/` | Авторский черновик, ожидает подачи на рассмотрение |
| **На рассмотрении** | `docs/reference/rfc/review/` | Открыто обсуждение сообщества и обратная связь |
| **Принято** | `docs/reference/rfc/accepted/` | Становится официальным документом дизайна, переходит в стадию реализации |
| **Отклонено** | `docs/reference/rfc/rejected/` | Отклонённые документы RFC |

## Подача RFC

1. Прочитайте [RFC_TEMPLATE.md](RFC_TEMPLATE.md) для ознакомления с требованиями к формату
2. Обратитесь к [EXAMPLE_full_feature_proposal.md](EXAMPLE_full_feature_proposal.md) для изучения стиля написания
3. Создайте новый файл с именем `порядковый_номер-описательное_название.md`
4. Поместите файл в каталог `docs/reference/rfc/draft/`
5. Обновите этот индексный файл, добавив новую запись RFC
6. Отправьте PR для прохождения процесса рассмотрения
