# Правила форматирования функций

---

## §4 Определение функций

**§4.1 Сигнатура функции.** Между именем функции и списком параметров пробел не ставится.

```
// ✅ Правильно
fn foo(a: Int, b: Int) -> Int { ... }

// ❌ Неправильно
fn foo (a: Int, b: Int) -> Int { ... }
```

**§4.2 Перенос списка параметров.** Если список параметров превышает ширину строки, каждый параметр записывается на отдельной строке с висячей запятой.

```
// При превышении ширины строки
fn very_long_function_name(first_param: Int, second_param: Int, third_param: Int) -> Int { ... }

// После форматирования
fn very_long_function_name(
    first_param: Int,
    second_param: Int,
    third_param: Int,
) -> Int { ... }
```

**§4.3 Тип возвращаемого значения.** Между типом возвращаемого значения и списком параметров ставится ` -> `, до и после `->` имеется по одному пробелу.

```
// ✅ Правильно
fn foo() -> Int { ... }

// ❌ Неправильно
fn foo()->Int { ... }
fn foo() ->Int { ... }
fn foo()-> Int { ... }
```

**§4.4 Тело функции.** Между телом функции и типом возвращаемого значения ставится один пробел.

```
// ✅ Правильно
fn foo() -> Int { 1 }

// ❌ Неправильно (два пробела)
fn foo() -> Int  { 1 }
```

## §7 Вызов функций

**§7.1 Список параметров.** Параметры разделяются запятыми, после запятой ставится пробел.

```
// ✅ Правильно
foo(1, 2, 3)

// ❌ Неправильно
foo(1,2,3)
foo(1 , 2 , 3)
```

**§7.2 Именованные параметры.** Именованные параметры используют формат `name = value`.

```
// ✅ Правильно
foo(x = 1, y = 2)

// ❌ Неправильно
foo(x=1, y=2)
```

**§7.3 Перенос параметров.** Когда список параметров превышает ширину строки, каждый параметр записывается на отдельной строке с висячей запятой.

```
// При превышении ширины строки
very_long_function_name(first_argument, second_argument, third_argument)

// После форматирования
very_long_function_name(
    first_argument,
    second_argument,
    third_argument,
)
```

## §12 Лямбда-выражения

**§12.1 Формат лямбда-выражения.** Лямбда-выражение использует формат `(params) => body`.

```
// ✅ Правильно
let f = (x) => x + 1;

// Тело с одним выражением
let f = (x) => x * 2;

// Тело с несколькими выражениями
let f = (x) => {
    let y = x + 1;
    y * 2
};
```
