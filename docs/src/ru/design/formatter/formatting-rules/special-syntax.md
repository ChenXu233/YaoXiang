# Специальные правила синтаксиса

---

## §13 F-String

**§13.1 Формат F-String.** F-String использует формат `f"..."`, интерполяция с помощью `{expr}`.

```
// ✅ Правильно
let msg = f"Hello, {name}!";
let msg = f"Result: {x + y}";
```

**§13.2 Спецификаторы формата.** F-String поддерживает спецификаторы формата `{expr:spec}`.

```
// ✅ Правильно
let msg = f"{value:.2f}";
```

## §14 Операторы импорта

**§14.1 Сортировка импортов.** Когда `sort_imports = true`, операторы импорта сортируются в следующем порядке:

1. Стандартная библиотека (`std`, `core`, `alloc`)
2. Внешние crates
3. Относительные пути (начинающиеся с `.` или `..`)

**§14.2 Сортировка внутри группы.** Импорты внутри одной группы сортируются по алфавиту.

```
// До сортировки
use z_crate;
use std::collections;
use a_crate;
use ./local;

// После сортировки
use std::collections;
use a_crate;
use z_crate;
use ./local;
```

## §17 Обработка ошибок

**§17.1 Оператор Try.** Использует формат `expr?`.

```
// ✅ Правильно
let x = foo()?;

// ❌ Неправильно
let x = foo() ?;
```

---

## §18 Блоки Unsafe

**§18.1 Формат Unsafe.** Использует формат `unsafe { ... }`.

```
// ✅ Правильно
let x = unsafe { dangerous_function() };

// ❌ Неправильно
let x = unsafe{ dangerous_function() };
```
