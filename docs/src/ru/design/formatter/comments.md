# Сохранение комментариев

---

## §C1 Типы комментариев

**§C1.1 Однострочные комментарии.** Начинаются с `//`, должны быть сохранены.

```
// Это однострочный комментарий
let x = 1;  // Это комментарий в конце строки
```

**§C1.2 Многострочные комментарии.** Заключены в `/* ... */`, поддерживают вложенность, должны быть сохранены.

```
/* Это многострочный комментарий */
/* Вложенный /* комментарий */ */
```

**§C1.3 Документационные комментарии.** Начинаются с `///`, должны быть сохранены.

```
/// Это документационный комментарий
fn foo() { ... }
```

---

## §C2 Расположение комментариев

**§C2.1 Комментарии в начале файла.** Комментарии в начале файла должны быть сохранены в начале файла.

**§C2.2 Межстрочные комментарии.** Комментарии между операторами должны быть сохранены на исходных позициях.

**§C2.3 Комментарии в конце строки.** Комментарии в конце строки должны быть сохранены в конце этой же строки.

---

## §C3 Сохранение пустых строк

**§C3.1 Сохранение пустых строк.** Пустые строки в исходном коде должны быть сохранены для разделения логических блоков.
