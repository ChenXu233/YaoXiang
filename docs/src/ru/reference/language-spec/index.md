# Спецификация языка

Документ со спецификацией синтаксиса и семантики языка YaoXiang.

## Содержание

- [Синтаксическая спецификация](./syntax.md) - Лексическая структура, синтаксические правила, приоритет операторов
- [Типовая система](./type-system.md) - Базовые типы, составные типы, generics, trait
- [Модульная система](./modules.md) - Определение модулей, импорт/экспорт, область видимости

- [Стандартная библиотека](./stdlib.md) - Базовая библиотека, библиотека IO, математическая библиотека

## Обзор

Данная спецификация определяет основной синтаксис и семантику языка программирования YaoXiang. Она является авторитетным справочником по языку и предназначена для разработчиков компиляторов и инструментов.

### Структура спецификации

1. **Синтаксическая спецификация**: Определяет лексическую структуру, синтаксические правила и приоритет операторов языка
2. **Типовая система**: Определяет типovую систему языка, включая базовые типы, составные типы, generics и trait
3. **Модульная система**: Определяет модульную систему языка, включая определение модулей, импорт/экспорт и область видимости
4. **Модель параллелизма**: Определяет модель параллелизма языка, включая асинхронное программирование, примитивы параллелизма и модель памяти
5. **Стандартная библиотека**: Определяет стандартную библиотеку языка, включая базовую библиотеку, библиотеку IO и математическую библиотеку
