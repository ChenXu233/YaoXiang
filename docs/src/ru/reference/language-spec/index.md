Документация по синтаксису и семантике языка YaoXiang.

## Содержание

- [Синтаксическая спецификация](./syntax.md) — лексическая структура, синтаксические правила, приоритет операторов

## Обзор

Данная спецификация определяет базовый синтаксис и семантику языка программирования YaoXiang. Она служит авторитетным справочником по языку и предназначена для разработчиков компиляторов и инструментов.

### Структура спецификации

1. **Синтаксическая спецификация**: определяет лексическую структуру языка, синтаксические правила и приоритет операторов
2. **Система типов**: определяет систему типов языка, включая базовые типы, составные типы, generics и trait
3. **Система модулей**: определяет систему модулей языка, включая определение модулей, импорт/экспорт и область видимости
4. **Модель конкурентности**: определяет модель конкурентности языка, включая асинхронное программирование, примитивы конкурентности и модель памяти
5. **Стандартная библиотека**: определяет стандартную библиотеку языка, включая базовую библиотеку, библиотеку ввода-вывода и математическую библиотеку
