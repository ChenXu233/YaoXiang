# Спецификация языка

Документ со спецификацией синтаксиса и семантики языка yaoxiang.

## Содержание

- [Спецификация синтаксиса](./syntax.md) — лексическая структура, синтаксические правила, приоритет операторов
- [Типовая система](./type-system.md) — базовые типы, составные типы, generics, trait
- [Система модулей](./modules.md) — определение модулей, импорт/экспорт, области видимости
- [Модель параллелизма](./concurrency.md) — семантика блоков {}, spawn примитивы параллелизма, обработка ошибок, типы ресурсов
- [Стандартная библиотека](./stdlib.md) — основная библиотека, библиотека IO, математическая библиотека

## Обзор

Данная спецификация определяет основной синтаксис и семантику языка программирования YaoXiang. Она служит авторитетным справочником по языку и предназначена для разработчиков компиляторов и инструментов.

### Структура спецификации

1. **Спецификация синтаксиса**: определяет лексическую структуру, синтаксические правила и приоритет операторов языка
2. **Типовая система**: определяет типовую систему языка, включая базовые типы, составные типы, generics и trait
3. **Система модулей**: определяет систему модулей языка, включая определение модулей, импорт/экспорт и области видимости
4. **Модель параллелизма**: определяет модель параллелизма языка, включая асинхронное программирование, примитивы параллелизма и модель памяти
5. **Стандартная библиотека**: определяет стандартную библиотеку языка, включая основную библиотеку, библиотеку IO и математическую библиотеку
