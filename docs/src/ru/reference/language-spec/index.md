# Спецификация языка

Документ, определяющий синтаксис и семантику языка программирования YaoXiang.

## Оглавление

- [Спецификация синтаксиса](./syntax.md) — лексическая структура, синтаксические правила, приоритет операторов
- [Типовая система](./type-system.md) — базовые типы, составные типы, generics, trait
- [Модульная система](./modules.md) — определение модулей, импорт/экспорт, области видимости
- [Модель конкурентности](./concurrency.md) — асинхронное программирование, примитивы конкурентности, модель памяти
- [Стандартная библиотека](./stdlib.md) — базовая библиотека, библиотека ввода-вывода, математическая библиотека

## Обзор

В данной спецификации определяются основной синтаксис и семантика языка программирования YaoXiang. Она служит авторитетным справочником по языку и предназначена для разработчиков компиляторов и инструментов.

### Структура спецификации

1. **Спецификация синтаксиса**: определяет лексическую структуру языка, синтаксические правила и приоритет операторов
2. **Типовая система**: определяет типovую систему языка, включая базовые типы, составные типы, generics и trait
3. **Модульная система**: определяет модульную систему языка, включая определение модулей, импорт/экспорт и области видимости
4. **Модель конкурентности**: определяет модель конкурентности языка, включая асинхронное программирование, примитивы конкурентности и модель памяти
5. **Стандартная библиотека**: определяет стандартную библиотеку языка, включая базовую библиотеку, библиотеку ввода-вывода и математическую библиотеку
