## Модель владения (сводка)

Полная модель описана в [Спецификации системы типов](./type-system.md), главы 11 и 12. Здесь приведена лишь сводка модели безопасности:

```yaoxiang
// Move (по умолчанию, нулевое копирование)
p2 = p

// ref (компилятор автоматически выбирает Rc или Arc)
shared = ref p

// clone (явное глубокое копирование)
p2 = p.clone()

// unsafe (голый указатель)
unsafe {
    ptr: *Point = &p
    (*ptr).x = 0.0
}
```

| Семантика | Описание | Накладные расходы |
|------|------|------|
| Move | По умолчанию, передача владения | Ноль |
| `&T` / `&mut T` | Заимствования令牌, доказательство прав компилятором нулевого размера | Ноль |
| `ref` | Компилятор автоматически выбирает Rc (внутри задачи) / Arc (между задачами) | По запросу |

## Синтаксис spawn (выдержка)

Полный синтаксис см. в [Спецификации синтаксиса](./syntax.md), раздел 3.10.

```yaoxiang
// spawn блок
(result_a, result_b) = spawn {
    parse(fetch("url1")),
    parse(fetch("url2"))
}

// spawn цикл
results = spawn for item in items {
    process(item)
}
```
