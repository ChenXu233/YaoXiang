# Коды ошибок

В этом документе перечислены ошибки, которые может возвращать менеджер пакетов YaoXiang, и способы их обработки.

## Список ошибок

### E0100: Проект уже существует

```
Error: Проект уже существует: <path>
```

**Причина**: Каталог проекта, который вы пытаетесь создать, уже существует.

**Способ устранения**:
1. Выберите другое имя проекта
2. Удалите или переместите существующий каталог

---

### E0101: Недействительный проект

```
Error: Не проект YaoXiang: yaoxiang.toml не найден
```

**Причина**: Текущий или указанный каталог не содержит файл `yaoxiang.toml`.

**Способ устранения**:
1. Убедитесь, что выполняете команду в каталоге проекта
2. Используйте `yaoxiang init` для создания нового проекта

---

### E0102: Зависимость не существует

```
Error: Зависимость не найдена: <name>
```

**Причина**: Вы пытаетесь выполнить операцию с несуществующей зависимостью.

**Способ устранения**:
1. Проверьте правильность написания имени зависимости
2. Используйте `yaoxiang list` для просмотра существующих зависимостей

---

### E0103: Зависимость уже существует

```
Error: Зависимость уже существует: <name>
```

**Причина**: Вы пытаетесь добавить уже существующую зависимость.

**Способ устранения**:
1. Если нужно обновить версию, сначала удалите с помощью `yaoxiang rm`
2. Или просто используйте существующую зависимость

---

### E0104: Неверный формат манифеста

```
Error: Неверный формат yaoxiang.toml: <details>
```

**Причина**: Файл `yaoxiang.toml` имеет неправильный формат.

**Способ устранения**:
1. Проверьте правильность TOML-синтаксиса
2. Убедитесь, что все обязательные поля присутствуют
3. Проверьте наличие синтаксических ошибок (отсутствие кавычек, запятых и т.д.)

---

### E0105: Ошибка ввода-вывода

```
Error: Ошибка ввода-вывода: <details>
```

**Причина**: Ошибка при чтении или записи файла.

**Возможные причины**:
- Недостаточно места на диске
- Недостаточно прав
- Файл занят другой программой

**Способ устранения**:
1. Проверьте свободное место на диске
2. Проверьте права доступа к файлам
3. Закройте другие программы, которые могут использовать файл

---

### E0106: Ошибка разбора TOML

```
Error: Ошибка разбора TOML: <details>
```

**Причина**: Ошибка формата TOML-файла.

**Способ устранения**:
1. Проверьте TOML-синтаксис
2. Проверьте правильность экранирования специальных символов

---

## Часто задаваемые вопросы

### В: Не удаётся установить зависимость. Что делать?

1. Проверьте сетевое подключение
2. Убедитесь в правильности имени и версии зависимости
3. Попробуйте выполнить `yaoxiang update` для обновления

### В: Возник конфликт версий. Что делать?

Проверьте в `yaoxiang.toml` наличие несовместимых требований к версиям зависимостей.

### В: Каталог vendor повреждён. Что делать?

Удалите каталог `vendor` и выполните `yaoxiang install` заново.
