# Формат yaoxiang.lock

`yaoxiang.lock` — это файл блокировки зависимостей YaoXiang, в котором записываются точные версии всех зависимостей.

## Обзор

- **Автоматическая генерация**: создаётся и обновляется автоматически при выполнении `yaoxiang install` и `yaoxiang update`
- **Не редактировать вручную**: этот файл обслуживается менеджером пакетов автоматически
- **Должен быть в системе контроля версий**: это гарантирует, что все члены команды и сборки CI используют одни и те же версии зависимостей

## Структура файла

```toml
# Автоматически сгенерировано менеджером пакетов YaoXiang

[package]
version = 1

[package.Имя_зависимости]
version = "номер_версии"
source = "тип_источника"
checksum = "контрольная_сумма (опционально)"
```

## Описание полей

### Секция [package]

| Поле | Тип | Описание |
|------|------|----------|
| `version` | integer | Версия формата файла блокировки, текущая — 1 |

### Секция [package.\<name\>]

| Поле | Тип | Описание |
|------|------|----------|
| `version` | string | Точная разрешённая версия |
| `source` | string | Источник зависимости: `registry`, `git`, `path` |
| `checksum` | string | SHA-256 контрольная сумма (опционально) |

## Пример

```toml
# Автоматически сгенерировано менеджером пакетов YaoXiang
# 2024-01-15 10:30:00

[package]
version = 1

[package.http]
version = "1.0.0"
source = "registry"
checksum = "abc123..."

[package.json]
version = "2.0.0"
source = "registry"

[package.utils]
version = "0.1.0"
source = "path"

[package.bleeding]
version = "main"
source = "git"
```

## Типы источников

| Тип | Описание | Пример конфигурации |
|------|----------|---------------------|
| `registry` | Получение из удалённого реестра | `http = "1.0.0"` |
| `git` | Получение из Git-репозитория | `{ git = "https://..." }` |
| `path` | Получение из локального пути | `{ path = "./lib" }` |

## Связь с manifest

```
yaoxiang.toml          yaoxiang.lock
     │                       │
     │   yaoxiang install    │
     ├──────────────────────►│
     │                       │
     │  объявляет "http = *>" │  блокирует "http = 1.0.0"
     │                       │
```

- `yaoxiang.toml`: объявляет **желаемые** зависимости (можно использовать диапазоны)
- `yaoxiang.lock`: записывает **фактически установленные** версии (точные версии)
