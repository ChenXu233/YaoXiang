# E2xxx：Семантический анализ

> Автогенерация из `src/util/diagnostic/codes/`

## Список ошибок

---

## E2001：Ошибка области видимости

**Категория**: Semantic

**Сообщение**: Переменная не находится в текущей области видимости

**Помощь**: Проверьте область видимости переменной и убедитесь, что она доступна здесь

---

## E2002：Повторное определение

**Категория**: Semantic

**Сообщение**: Переменная определена несколько раз в одной области видимости

**Помощь**: Переименуйте или удалите повторное определение

---

## E2003：Ошибка владения

**Категория**: Semantic

**Сообщение**: Ограничение владения не удовлетворено

**Помощь**: Проверьте семантику владения для данного значения

---

## E2010：Неизменяемое присваивание

**Категория**: Semantic

**Сообщение**: Попытка изменить неизменяемую переменную

**Помощь**: Используйте `mut` для объявления изменяемой переменной

---

## E2011：Использование неинициализированной переменной

**Категория**: Semantic

**Сообщение**: Использование неинициализированной переменной

**Помощь**: Инициализируйте переменную перед её использованием

---

## E2012：Конфликт изменяемости

**Категория**: Semantic

**Сообщение**: Использование изменяемой ссылки в неизменяемом контексте

**Помощь**: Убедитесь, что изменяемость ссылки соответствует контексту использования
