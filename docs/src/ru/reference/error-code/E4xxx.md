# E4xxx：Обобщения и Traits

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E4001：Нарушение ограничения обобщения

**Категория**: Generic

**Сообщение**: Тип не удовлетворяет ограничению обобщения

**Помощь**: Убедитесь, что тип удовлетворяет всем необходимым ограничениям traits

---

## E4002：Trait не найден

**Категория**: Generic

**Сообщение**: Ссылаемая trait не существует

**Помощь**: Проверьте имя trait или импортируйте корректную trait

---

## E4003：Реализация trait отсутствует

**Категория**: Generic

**Сообщение**: Тип не реализует необходимую trait

**Помощь**: Добавьте реализацию trait для этого типа

---

## E4004：Конфликтующие реализации traits

**Категория**: Generic

**Сообщение**: Несколько реализаций traits конфликтуют

**Помощь**: Разрешите конфликтующие реализации
