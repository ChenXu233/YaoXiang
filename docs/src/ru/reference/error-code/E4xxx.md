# E4xxx：Generics и Traits

> Автогенерация из `src/util/diagnostic/codes/`

## Список ошибок

## E4001：Нарушение generic-ограничения

**Категория**: Generic

**Сообщение**: Тип не удовлетворяет generic-ограничению

**Помощь**: Убедитесь, что тип удовлетворяет всем необходимым trait bounds

---

## E4002：Trait не найден

**Категория**: Generic

**Сообщение**: Указанный trait не существует

**Помощь**: Проверьте имя trait или импортируйте корректный trait

---

## E4003：Реализация trait отсутствует

**Категория**: Generic

**Сообщение**: Тип не реализует требуемый trait

**Помощь**: Добавьте реализацию trait для этого типа

---

## E4004：Конфликтующие реализации trait

**Категория**: Generic

**Сообщение**: Несколько реализаций trait конфликтуют

**Помощь**: Разрешите конфликтующие реализации
