# E4xxx: 泛型与特质

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E4001: Нарушение ограничения типа

**Категория**: 泛型

**Сообщение**: Тип не удовлетворяет ограничению типа

**Справка**: Убедитесь, что тип удовлетворяет всем необходимым ограничениям типажа

---

## E4002: 类型未找到

**Категория**: 泛型

**Сообщение**: Указанный типаж не существует

**Справка**: Проверьте имя типажа или импортируйте правильный типаж

---

## E4003: Отсутствует реализация типажа

**Категория**: 泛型

**Сообщение**: Тип не реализует требуемый типаж

**Справка**: Добавьте реализацию типажа для этого типа

---

## E4004: Конфликтующие реализации типажа

**Категория**: 泛型

**Сообщение**: Несколько реализаций типажа конфликтуют

**Справка**: Разрешите конфликтующие реализации
