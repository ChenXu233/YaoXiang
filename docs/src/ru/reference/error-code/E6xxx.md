# E6xxx：Ошибки времени выполнения

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E6001：Деление на ноль

**Категория**: Runtime

**Сообщение**: Попытка деления на ноль

**Помощь**: Добавьте проверку для предотвращения деления на ноль

---

## E6002：Разыменование нулевого указателя

**Категория**: Runtime

**Сообщение**: Попытка доступа к пустому значению

**Помощь**: Добавьте проверку на пустое значение перед доступом

---

## E6003：Выход индекса массива за границы

**Категория**: Runtime

**Сообщение**: Индекс массива вышел за границы во время выполнения

**Помощь**: Убедитесь, что индекс находится в пределах границ массива

---

## E6004：Переполнение стека

**Категория**: Runtime

**Сообщение**: Превышен лимит глубины рекурсии стека

**Помощь**: Уменьшите глубину рекурсии или используйте итерацию

---

## E6005：Проверка assert не прошла

**Категория**: Runtime

**Сообщение**: Проверка assert не прошла во время выполнения

**Помощь**: Исправьте условие проверки или предоставьте корректные входные данные
