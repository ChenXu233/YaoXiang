# E5xxx：Модули и импорт

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E5001：Модуль не найден

**Категория**: Module

**Сообщение**: Указанный модуль не существует

**Помощь**: Проверьте путь к модулю и убедитесь, что он существует

---

## E5002：Ошибка импорта

**Категория**: Module

**Сообщение**: Не удалось импортировать модуль

**Помощь**: Проверьте путь импорта и разрешение модуля

---

## E5003：Экспорт не найден

**Категория**: Module

**Сообщение**: Указанный экспорт не существует в модуле

**Помощь**: Проверьте экспорты модуля

---

## E5004：Циклическая зависимость

**Категория**: Module

**Сообщение**: Модули имеют циклические зависимости

**Помощь**: Выполните рефакторинг для устранения циклической зависимости
