# E8xxx: Внутренние ошибки компилятора

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E8001: Internal compiler error (Внутренняя ошибка компилятора)

**Категория**: Internal (Внутренняя)

**Сообщение**: An internal error occurred in the compiler (Произошла внутренняя ошибка в компиляторе)

**Помощь**: Please report this bug at https://github.com/yaoxiang-lang/yaoxiang/issues (Пожалуйста, сообщите об этой ошибке на https://github.com/yaoxiang-lang/yaoxiang/issues)

---

## E8002: Unexpected panic (Неожиданная паника)

**Категория**: Internal (Внутренняя)

**Сообщение**: The compiler encountered an unexpected panic (Компилятор столкнулся с неожиданной паникой)

**Помощь**: Please report this bug with the panic details (Пожалуйста, сообщите об этой ошибке вместе с деталями паники)

---

## E8003: Compiler phase error (Ошибка фазы компилятора)

**Категория**: Internal (Внутренняя)

**Сообщение**: An error occurred during a compiler phase (Произошла ошибка во время фазы компиляции)
