# E7xxx: Ошибки ввода-вывода и системы

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E7001: Файл не найден

**Категория**: Ввод-вывод

**Сообщение**: Указанный файл не существует

**Помощь**: Проверьте путь к файлу и убедитесь, что файл существует

---

## E7002: Доступ запрещён

**Категория**: Ввод-вывод

**Сообщение**: Недостаточно прав для доступа к файлу

**Помощь**: Проверьте права доступа к файлу или запустите программу с соответствующими привилегиями

---

## E7003: Ошибка ввода-вывода

**Категория**: Ввод-вывод

**Сообщение**: Произошла неопределённая ошибка ввода-вывода

**Помощь**: Подробнее об ошибке можно узнать в деталях сообщения

---

## E7004: Ошибка сети

**Категория**: Ввод-вывод

**Сообщение**: Произошла ошибка сети

**Помощь**: Проверьте сетевое подключение и повторите попытку
