# E7xxx：Ошибки ввода-вывода и системы

> Автоматически сгенерировано из `src/util/diagnostic/codes/`

## Список ошибок

## E7001：File not found

**Категория**: Ввод-вывод

**Сообщение**: Указанный файл не существует

**Справка**: Проверьте путь к файлу и убедитесь, что файл существует

---

## E7002：Permission denied

**Категория**: Ввод-вывод

**Сообщение**: Недостаточно прав для доступа к файлу

**Справка**: Проверьте права доступа к файлу или запустите с соответствующими привилегиями

---

## E7003：I/O error

**Категория**: Ввод-вывод

**Сообщение**: Произошла неопределённая ошибка ввода-вывода

**Справка**: Подробнее см. в описании ошибки

---

## E7004：Network error

**Категория**: Ввод-вывод

**Сообщение**: Произошла сетевая ошибка

**Справка**: Проверьте сетевое подключение и повторите попытку

---
