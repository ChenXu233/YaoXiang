## Встроенные типы

### Примитивные типы

| Тип | Описание | Пример |
|------|------|------|
| `Void` | Пустое значение/отсутствие возвращаемого значения | `()` |
| `Bool` | Логическое значение | `true`, `false` |
| `Int` | Целое число | `42`, `-10` |
| `Float` | Число с плавающей точкой | `3.14`, `-0.5` |
| `Char` | Символ | `'a'`, `'中'` |
| `String` | Строка | `"hello"` |

### Составные типы

| Тип | Описание | Пример |
|------|------|------|
| `List(T)` | Список однотипных элементов | `[1, 2, 3]` |
| `Tuple(T1, T2, ...)` | Кортеж разнотипных элементов | `(1, "hello")` |
| `Dict(K, V)` | Отображение ключ-значение | `{"a": 1}` |
| `(Args) -> Ret` | Функциональный тип | `(Int) -> Int` |

### Определяемые пользователем типы

```yaoxiang
// Запись (структура)
Point: Type = { x: Float, y: Float }

// Перечисление
Result: (T: Type, E: Type) -> Type = { ok(T) | err(E) }

// Интерфейс (все поля — функции)
Callable: Type = { call: (String) -> Void }
```

## Встроенные функции

### Вывод

```yaoxiang
print(value)           // печать без перевода строки
println(value)         // печать с переводом строки
```

### Преобразование

```yaoxiang
to_string(value)       // преобразовать в строку
to_int(value)          // преобразовать в целое число
to_float(value)        // преобразовать в число с плавающей точкой
```

### Проверка типов

```yaoxiang
typeof(value)         // возвращает имя типа
is_type(value, type)  // проверяет тип
```

## Ключевые слова

| Ключевое слово | Описание |
|--------|------|
| `Type` | Мета-тип |
| `spawn` | Помечает spawn-функцию |
| `spawn for` | Параллельный цикл |
| `spawn {}` | spawn-блок |
| `if` / `elif` / `else` | Ветвление условия |
| `match` | Сопоставление с образцом |
| `while` / `for` | Циклы |
| `return` | Возврат значения |
| `ref` | Создание ссылки |
| `mut` | Метка изменяемости |

## Краткий справочник по синтаксису

### Объявление переменных

```yaoxiang
// Неизменяемая переменная (по умолчанию)
x: Int = 42
y = 42                 // вывод типа

// Изменяемая переменная
mut count: Int = 0
count = count + 1
```

### Определение функций

```yaoxiang
// Обычная функция
add: (a: Int, b: Int) -> Int = a + b

// spawn-функция (автоматически параллельная)
fetch: (url: String) -> JSON spawn = HTTP.get(url).json()

// Обобщённая функция
identity: [T](x: T) -> T = x
```

### Управление потоком

```yaoxiang
// Условие
if x > 0 {
    print("positive")
} elif x < 0 {
    print("negative")
} else {
    print("zero")
}

// Сопоставление с образцом
match result {
    ok(value) => print("success: " + value),
    err(error) => print("error: " + error),
}

// Цикл
for i in 0..10 {
    print(i)
}
```

### Обработка ошибок

```yaoxiang
// оператор ? для распространения ошибки
data = fetch_file(path)?
```

## Приоритет операторов

| Приоритет | Оператор |
|--------|--------|
| Высший | `( )` вызов функции |
| | `.` доступ к полю |
| | `[ ]` индексация |
| | `unary -` унарный минус |
| | `* / %` умножение/деление/остаток |
| | `+ -` сложение/вычитание |
| | `== != < > <= >=` сравнение |
| | `and or` логические операции |
| Низший | `=` присваивание |

## Полный пример

```yaoxiang
// Вычисление чисел Фибоначчи
fib: (n: Int) -> Int = if n <= 1 {
    n
} else {
    fib(n - 1) + fib(n - 2)
}

// Главная функция
main: () -> Void = {
    print("Fibonacci(10) = " + fib(10).to_string())
}
```

## Руководство по участию

Стандартная библиотека находится в разработке, приветствуется вклад!

1. Выберите модуль (например, `std.io`, `std.net`)
2. Реализуйте функции в `src/std/`
3. Добавьте документационные комментарии
4. Отправьте PR
