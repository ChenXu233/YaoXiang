## Встроенные типы

### Примитивные типы

| Тип | Описание | Пример |
|-----|----------|--------|
| `Void` | пустое значение / отсутствие возвращаемого значения | `()` |
| `Bool` | Логическое значение | `true`, `false` |
| `Int` | Целое число | `42`, `-10` |
| `Float` | Число с плавающей точкой | `3.14`, `-0.5` |
| `Char` | Символ | `'a'`, `'中'` |
| `String` | Строка | `"hello"` |

### Составные типы

| Тип | Описание | Пример |
|-----|----------|--------|
| `List[T]` | Список однотипных элементов | `[1, 2, 3]` |
| `Tuple(T1, T2, ...)` | Кортеж разнотипных элементов | `(1, "hello")` |
| `Dict[K, V]` | Отображение ключ-значение | `{"a": 1}` |
| `Fn(Args) -> Ret` | Тип функции | `(Int) -> Int` |

### Пользовательские типы

```yaoxiang
// Запись типа (структура)
Point: Type = { x: Float, y: Float }

// Перечисление типа
Result: Type[T, E] = { ok(T) | err(E) }

// Интерфейсный тип (все поля — функции)
Callable: Type = { call: (String) -> Void }
```

## Встроенные функции

### Вывод

```yaoxiang
print(value)           // Печать без перевода строки
println(value)         // Печать с переводом строки
```

### Преобразование

```yaoxiang
to_string(value)       // Преобразовать в строку
to_int(value)          // Преобразовать в целое число
to_float(value)        // Преобразовать в число с плавающей точкой
```

### Проверка типов

```yaoxiang
typeof(value)         // Возвращает имя типа
is_type(value, type)  // Проверяет тип
```

## Краткий справочник по синтаксису

### Объявление переменных

```yaoxiang
// Неизменяемая переменная (по умолчанию)
x: Int = 42
y = 42                 // Вывод типа

// Изменяемая переменная
mut count: Int = 0
count = count + 1
```

### Определение функций

```yaoxiang
// Обычная функция
add: (a: Int, b: Int) -> Int = a + b

// 并作函数 (автоматический параллелизм)
fetch: (url: String) -> JSON spawn = HTTP.get(url).json()

// Обобщённая функция
identity: [T](x: T) -> T = x
```

### Управление потоком выполнения

```yaoxiang
// Условие
if x > 0 {
    println("positive")
} elif x < 0 {
    println("negative")
} else {
    println("zero")
}

// Паттерн матчинг
match result {
    ok(value) => println("success: " + value),
    err(error) => println("error: " + error),
}

// Цикл
for i in 0..10 {
    print(i)
}
```

### Обработка ошибок

```yaoxiang
// Оператор ? распространяет ошибку
data = fetch_file(path)?
```

## Полный пример

```yaoxiang
// Вычисление последовательности Фибоначчи
fib: (n: Int) -> Int = if n <= 1 {
    n
} else {
    fib(n - 1) + fib(n - 2)
}

// Главная функция
main: () -> Void = {
    println("Fibonacci(10) = " + fib(10).to_string())
}
```

## Руководство по внесению вклада

Стандартная библиотека в разработке, вклад приветствуется!

1. Выберите модуль (например, `std.io`, `std.net`)
2. Реализуйте функции в `src/std/`
3. Добавьте документационные комментарии
4. Отправьте PR
