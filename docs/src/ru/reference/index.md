## Встроенные типы

### Примитивные типы

| Тип | Описание | Пример |
|------|------|------|
| `Void` | Пустое значение/отсутствие возвращаемого значения | `()` |
| `Bool` | Логическое значение | `true`, `false` |
| `Int` | Целое число | `42`, `-10` |
| `Float` | Число с плавающей запятой | `3.14`, `-0.5` |
| `Char` | Символ | `'a'`, `'中'` |
| `String` | Строка | `"hello"` |

### Составные типы

| Тип | Описание | Пример |
|------|------|------|
| `List[T]` | Список элементов одного типа | `[1, 2, 3]` |
| `Tuple(T1, T2, ...)` | Кортеж разнородных элементов | `(1, "hello")` |
| `Dict[K, V]` | Отображение ключ-значение | `{"a": 1}` |
| `Fn(Args) -> Ret` | Тип функции | `(Int) -> Int` |

### Типы, определяемые пользователем

```yaoxiang
// Record type (структура)
Point: Type = { x: Float, y: Float }

// Enum type
Result: Type[T, E] = { ok(T) | err(E) }

// Interface type (все поля являются функциями)
Callable: Type = { call: (String) -> Void }
```

## Встроенные функции

### Вывод

```yaoxiang
print(value)           // Печать без перевода строки
println(value)         // Печать с переводом строки
```

### Преобразование

```yaoxiang
to_string(value)       // Преобразование в строку
to_int(value)          // Преобразование в целое число
to_float(value)        // Преобразование в число с плавающей запятой
```

### Проверка типа

```yaoxiang
typeof(value)         // Возвращает название типа
is_type(value, type)  // Проверяет тип
```

| `mut` | Изменяемый маркер |

## Краткий справочник по синтаксису

### Объявление переменных

```yaoxiang
// Неизменяемая переменная (по умолчанию)
x: Int = 42
y = 42                 // Вывод типа

// Изменяемая переменная
mut count: Int = 0
count = count + 1
```

### Определение функции

```yaoxiang
// Обычная функция
add: (a: Int, b: Int) -> Int = a + b

// Spawn функция (автоматическая конкурентность)
fetch: (url: String) -> JSON spawn = HTTP.get(url).json()

// Обобщённая функция
identity: [T](x: T) -> T = x
```

### Управление потоком

```yaoxiang
// Условие
if x > 0 {
    println("positive")
} elif x < 0 {
    println("negative")
} else {
    println("zero")
}

// Сопоставление с образцом
match result {
    ok(value) => println("success: " + value),
    err(error) => println("error: " + error),
}

// Цикл
for i in 0..10 {
    print(i)
}
```

### Обработка ошибок

```yaoxiang
// Оператор ? для распространения ошибки
data = fetch_file(path)?
```

## Приоритет операторов

| Приоритет | Оператор |
|--------|--------|
| Самый высокий | `( )` Вызов функции |
| | `.` Доступ к полю |
| | `[ ]` Индекс |
| | `unary -` Унарный минус |
| | `* / %` Умножение, деление, остаток |
| | `+ -` Сложение, вычитание |
| | `== != < > <= >=` Сравнение |
| | `and or` Логические операции |
| Самый низкий | `=` Присваивание |

## Полный пример

```yaoxiang
// Вычисление последовательности Фибоначчи
fib: (n: Int) -> Int = if n <= 1 {
    n
} else {
    fib(n - 1) + fib(n - 2)
}

// Главная функция
main: () -> Void = {
    println("Fibonacci(10) = " + fib(10).to_string())
}
```
