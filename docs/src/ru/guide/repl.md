# REPL — интерактивный интерпретатор

REPL YaoXiang (Read-Eval-Print Loop) — это интерактивная среда выполнения кода, которая позволяет вводить и выполнять код YaoXiang построчно. Она идеально подходит для обучения, тестирования и отладки.

## Быстрый старт

### Запуск REPL

Для запуска REPL выполните в терминале следующую команду:

```bash
yaoxiang repl
```

Или запустите `yaoxiang` без аргументов:

```bash
yaoxiang
```

После запуска вы увидите приглашение:

```
YaoXiang REPL - Type :help for assistance
Press Ctrl+D or :quit to exit

>>
```

### Основы использования

Введите код YaoXiang после приглашения `>>` и нажмите Enter для выполнения:

```rust
>> 1 + 2
3

>> "Hello, World!"
"Hello, World!"

>> let x = 10
>> x * 2
20
```

### Выход из REPL

Существует три способа выйти из REPL:

1. **Горячая клавиша**: нажмите `Ctrl+D`
2. **Команда**: введите `:quit` или `:q`
3. **Прерывание**: нажмите `Ctrl+C` для прерывания текущего ввода

## Система команд

REPL предоставляет набор специальных команд, начинающихся с двоеточия `:`.

### Команда помощи

```rust
>> :help
```

Отображает справочную информацию обо всех доступных командах.

### Команда выхода

```rust
>> :quit
```

Выход из REPL. Можно также использовать сокращение `:q`.

### Команда очистки

```rust
>> :clear
```

Очищает все определённые переменные и функции, сбрасывает состояние REPL. Можно также использовать сокращение `:c`.

### Команда просмотра типа

```rust
>> :type x
```

Показывает информацию о типе символа `x`. Можно также использовать сокращение `:t`.

**Пример**:

```rust
>> let name = "YaoXiang"
>> :type name
name: String

>> fn add(a: Int, b: Int) -> Int = a + b
>> :type add
add: fn(Int, Int) -> Int
```

### Команда списка символов

```rust
>> :symbols
```

Выводит список всех определённых в текущем сеансе REPL символов (переменных и функций). Можно также использовать сокращения `:i` или `:info`.

**Пример**:

```rust
>> let x = 10
>> let y = 20
>> fn greet(name: String) -> String = "Hello, " + name
>> :symbols
x: Int
y: Int
greet: fn(String) -> String
```

### Команда истории

```rust
>> :history
```

Отображает историю введённых команд. Можно также использовать сокращение `:hist`.

### Команда статистики

```rust
>> :stats
```

Показывает статистику выполнения, включая количество вычислений и общее время выполнения.

**Пример**:

```rust
>> :stats
Eval count: 5
Total time: 12.34ms
```

## Выполнение кода

### Выполнение выражений

REPL может выполнять любые корректные выражения YaoXiang:

```rust
>> 1 + 2
3

>> 10 * 5 + 3
53

>> "Hello" + " " + "World"
"Hello World"

>> true && false
false
```

### Определение переменных

Для определения переменных используется ключевое слово `let`:

```rust
>> let name = "YaoXiang"
>> let age = 25
>> let pi = 3.14159
```

После определения переменные можно использовать в последующем коде:

```rust
>> name
"YaoXiang"

>> age + 5
30
```

### Определение функций

Для определения функций используется ключевое слово `fn`:

```rust
>> fn add(a: Int, b: Int) -> Int = a + b
>> fn greet(name: String) -> String = "Hello, " + name
```

Вызов функций:

```rust
>> add(3, 4)
7

>> greet("World")
"Hello World"
```

### Многострочный код

REPL поддерживает ввод многострочного кода. При обнаружении незавершённого кода (например, незакрытых скобок) автоматически активируется режим продолжения строки:

```rust
>> fn factorial(n: Int) -> Int =
..   if n <= 1 then 1
..   else n * factorial(n - 1)
```

Приглашение продолжения строки — `..`, указывающее на режим многострочного ввода.

### Определение структур

```rust
>> struct Point {
..   x: Float,
..   y: Float
.. }
```

### Определение перечислений

```rust
>> enum Color {
..   Red,
..   Green,
..   Blue
.. }
```

## Автодополнение

REPL предоставляет интеллектуальную функцию автодополнения, которая помогает быстрее вводить код.

### Способы активации

Нажмите клавишу `Tab` для вызова автодополнения.

### Что дополняется

1. **Дополнение ключевых слов**: ключевые слова языка YaoXiang
   - `let`, `fn`, `if`, `else`, `match`, `for`, `while`, `return` и другие

2. **Дополнение переменных**: уже определённые переменные
   - Введите первые символы имени переменной, нажмите Tab для дополнения

3. **Дополнение функций**: уже определённые функции
   - Введите первые символы имени функции, нажмите Tab для дополнения

4. **Дополнение встроенных функций**: встроенные функции
   - `print`, `len`, `range`, `typeof`, `assert` и другие

### Примеры дополнения

```rust
>> let my_variable = 42
>> my_<Tab>
my_variable: Int

>> fn calculate_sum(a: Int, b: Int) -> Int = a + b
>> calc<Tab>
calculate_sum: fn(Int, Int) -> Int
```

## Продвинутые функции

### Обработка ошибок

При возникновении ошибок в коде REPL отображает подробную информацию:

```rust
>> let x = 10 / 0
Error: Runtime error: DivisionByZero

>> undefined_variable
Error: Unknown symbol: undefined_variable
```

Ошибки не прерывают сеанс REPL — вы можете продолжать вводить новый код.

### История команд

REPL автоматически сохраняет историю команд, поддерживая:

- **Стрелки вверх/вниз**: просмотр истории команд
- **Поиск**: введите часть текста и используйте стрелки вверх/вниз для поиска
- **Файл истории**: история сохраняется в файле и автоматически загружается при следующем запуске

### Статистика выполнения

Используйте команду `:stats` для просмотра статистики выполнения:

```rust
>> :stats
Eval count: 15
Total time: 45.67ms
```

Это помогает отслеживать производительность кода.

## Лучшие практики

### 1. Используйте осмысленные имена переменных

```rust
// Хорошо
let user_name = "YaoXiang"
let max_retries = 3

// Плохо
let x = "YaoXiang"
let n = 3
```

### 2. Определяйте функции для повторного использования кода

```rust
>> fn is_even(n: Int) -> Bool = n % 2 == 0
>> is_even(4)
true
>> is_even(7)
false
```

### 3. Используйте `:clear` для сброса состояния

Когда состояние REPL становится запутанным, используйте `:clear`:

```rust
>> :clear
Context cleared
```

### 4. Используйте автодополнение для повышения эффективности

Введите первые символы и нажмите Tab для быстрого дополнения имён переменных и функций.

### 5. Используйте многострочный ввод для сложного кода

```rust
>> fn fibonacci(n: Int) -> Int =
..   if n <= 1 then n
..   else fibonacci(n - 1) + fibonacci(n - 2)
```

## Часто задаваемые вопросы

### В: Как просмотреть определение функции?

О: Используйте команду `:type` для просмотра сигнатуры функции:

```rust
>> :type my_function
my_function: fn(Int, String) -> Bool
```

### В: Как очистить все определения?

О: Используйте команду `:clear`:

```rust
>> :clear
```

### В: Почему мой многострочный код не выполняется?

О: Проверьте наличие незакрытых скобок, кавычек или фигурных скобок. REPL ожидает полного ввода кода.

### В: Как прервать долго выполняющийся код?

О: Нажмите `Ctrl+C` для прерывания текущего выполнения.

### В: Какие типы данных поддерживает REPL?

О: REPL поддерживает все типы данных YaoXiang:
- `Int`: целое число
- `Float`: число с плавающей запятой
- `String`: строка
- `Bool`: логическое значение
- `Unit`: тип Unit (пустой тип)
- пользовательские структуры и перечисления
