# Руководство по содействию

Мы приветствуем вклад в проект YaoXiang в виде кода и документации!

## Как внести вклад

1. **Fork** репозитория
2. Создайте ветку для функциональности: `git checkout -b feature/xxx`
3. Зафиксируйте изменения: `git commit -m "feat: xxx"`
4. Отправьте ветку: `git push origin feature/xxx`
5. Создайте **Pull Request**

## Стандарты кода

- Следуйте стилю кода проекта
- Убедитесь, что тесты проходят
- Обновите соответствующую документацию

## Стандарты коммитов

```
feat: новая функциональность
fix: исправление ошибки
docs: обновление документации
refactor: рефакторинг
test: связанное с тестированием
chore: обновление сборки/инструментов
```
