# Проверщик заимствований (Lifetime)

> **Состояние модуля**: стабильный (4 улучшения в очереди)
> **Расположение**: `src/middle/passes/lifetime/`
> **Последнее обновление**: 2026-06-01

---

## Обзор модуля

Модуль проверщика заимствований представляет собой полноценную систему анализа **владения и управления временем жизни**, отвечающую за проверку семантики Move, конфликтов заимствования, нарушений изменяемости и других проблем, связанных с владением.

**Объём кода**: около 300 КБ исходного кода (15 подфайлов)

---

## Сравнение с RFC (RFC-009 модель владения)

| Пункты дизайна RFC | Состояние реализации | Пояснение |
|-------------------|---------------------|-----------|
| Семантика Move (по умолчанию) | ✅ Реализовано | MoveChecker обнаруживает UseAfterMove |
| Токены заимствования &T/&mut T | ✅ Реализовано | BorrowChecker реализует обнаружение конфликтов токенов |
| &T копируемо (Dup) | ✅ Реализовано | Несколько токенов &T могут сосуществовать |
| &mut T линейно | ✅ Реализовано | Активный &mut T из одного источника может быть только один |
| Обнаружение конфликтов токенов (flow-sensitive liveness analysis) | ✅ Реализовано | Отслеживание состояния токенов внутри тела функции |
| Ключевое слово ref (автоматический выбор Rc/Arc) | ⚠️ Частично реализовано | Проверщик семантики ref существует |
| clone() явное глубокое копирование | ✅ Реализовано | CloneChecker обнаруживает clone перемещённых/освобождённых значений |
| unsafe + *T | ✅ Реализовано | UnsafeChecker проверяет операции с голыми указателями за пределами блока unsafe |
| Внутризадачные циклы: тихое разрешение | ✅ Реализовано | IntraTaskCycleTracker отслеживает в режиме предупреждений |
| Межзадачные циклы: lint | ✅ Реализовано | CycleChecker обнаруживает межзадачные циклические ссылки |
| Без времени жизни 'a | ✅ Соответствует дизайну | Реализация не содержит параметров времени жизни |
| Ограничения Send/Sync | ✅ Реализовано | SendSyncChecker работает независимо от OwnershipChecker |
