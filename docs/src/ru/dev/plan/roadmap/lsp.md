## Обзор модуля

Модуль LSP реализует Language Server Protocol, обеспечивая интеллектуальные функции для редакторов/IDE. Область реализации охватывает этапы 0-5 RFC-017, с досрочной реализацией расширенных функций: семантические токены, переименование, кодовые операции и другие.

**Объём кода**：7 542 строки (21 файл Rust)

---
