# Форматтер（Formatter）

> **Статус модуля**: стабильный (0 пунктов к улучшению)
> **Расположение**: `src/formatter/`
> **Последнее обновление**: 2026-06-04

---

## Обзор модуля

Форматтер отвечает за автоматическое форматирование исходного кода YaoXiang. Поддерживает полное форматирование узлов AST, включая выражения, операторы, типы и т.д.

**Объём кода**: 2 397 строк (19 исходных файлов)

---
