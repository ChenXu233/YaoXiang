# Форматировщик (Formatter)

> **Статус модуля**: стабильный (0 элементов в очереди на улучшение)

## Обзор модуля

Форматировщик отвечает за автоматическое форматирование исходного кода YaoXiang. Поддерживает полное форматирование узлов AST, включая выражения, операторы, типы и другие элементы.

**Объём кода**: 2 397 строк (19 исходных файлов)
