# Форматировщик (Formatter)

> **Статус модуля**: Базовая реализация завершена
> **Расположение**: `src/formatter/`
> **Последнее обновление**: 2026-06-01

## Обзор модуля

Форматировщик отвечает за автоматическое форматирование исходного кода YaoXiang. Поддерживает полное форматирование AST-узлов, включая выражения, операторы, типы и другие элементы.

**Объём кода**: 2 397 строк (19 файлов исходного кода)

## Не полностью реализованные спецификации

| Спецификация | Статус | Описание расхождений |
|------|------|----------|
| §2.2 Приоритет стратегий переноса | ⚠️ Частично реализовано | Реализован только перенос binop, полная цепочка приоритетов не реализована |
| §4.2 Перенос списков параметров (trailing comma) | ⚠️ Частично реализовано | Форматирование списков параметров всегда выводится в одну строку, автоматический перенос не выполняется |
| §6.2 Однострочные блоки кода | ❌ Не реализовано | Всегда выводится многстрочный формат `{ stmt }` |
| §6.3 Пустые блоки кода | ⚠️ Не полностью реализовано | Для пустых блоков выводится `{\n}` (две строки), а не `{}` |
| §8.3 Режим одинарных кавычек | ❌ Не реализовано | Поле `single_quote` существует, но не читается |
| §E2 Коды завершения | ❌ Не реализовано | Специфичные коды завершения не используются согласно спецификации |
