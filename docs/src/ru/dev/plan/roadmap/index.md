> **Последнее обновление**：2026-06-04

---

## Обзор проекта

YaoXiang — это язык программирования, ориентированный на будущее, с архитектурой компиляции в байт-код + интерпретация через VM. В настоящее время находится на стадии v0.7.x, основной компилятор и среда выполнения в основном завершены.

## Основные выводы

### Завершённые области

1. **Фронтенд компилятора**：лексический анализ, парсинг, проверка типов полностью завершены, отличное покрытие тестами
2. **Система владения**：полный проверяльщик заимствований, 14/15 подмодулей полностью функциональны
3. **Инструментарий**：LSP, менеджер пакетов, форматтер в основном завершены
4. **Сайт документации**：многоязычная поддержка, автоматическое развёртывание CI/CD

### Основные пробелы

1. **Модель 并作**（RFC-001）：DAG анализатор зависимостей не реализован, аннотации 并作 управления неполны
2. **Трёхуровневая архитектура среды выполнения**（RFC-008）：завершён только этап A, LLVM AOT бэкенд не реализован
3. **Зависимые от значения типы**（RFC-011）：движок вычислений времени компиляции не реализован
4. **Система конфигурации**（RFC-015）：пользовательская конфигурация не реализована
5. **Тесты стандартной библиотеки**：только 8 юнит-тестов, крайне недостаточно
