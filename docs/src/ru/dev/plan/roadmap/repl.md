## Обзор модуля

Модуль REPL (Read-Eval-Print Loop) предоставляет интерактивную среду программирования. Использует абстракцию на основе trait, поддерживает различные реализации backend.

## Статус интеграции

REPL интегрирован в следующие компоненты：

1. **DevShell**（`src/backends/dev/shell.rs`）：переключение в режим REPL через команду `:repl`
2. **Экспорт модуля**（`src/backends/dev/mod.rs`）：экспорт `SessionREPL`, `Evaluator`, `REPLBackend`
3. **CLI входная точка**：запуск через `yaoxiang repl` или `yaoxiang`
