# 控制流

在上一章你学会了如何定义函数。现在我们来学习如何让代码拥有"做决策"和"重复执行"的能力——这就是**控制流**。

YaoXiang 提供了五种控制流结构，各司其职：

| 控制流 | 用途 | 一句话描述 |
|--------|------|------------|
| `if-elif-else` | 条件判断 | 根据条件选择执行路径 |
| `for` | 遍历循环 | 逐个处理集合中的每个元素 |
| `while` | 条件循环 | 只要条件成立就一直循环 |
| `break` / `continue` | 循环控制 | 提前跳出循环或跳过当前迭代 |
| `match` | 模式匹配 | 根据值的结构分支处理 |

快速感受一下：

```yaoxiang
// if 是表达式，可以返回值
status = if score >= 60 { "及格" } else { "不及格" }

// for 遍历范围
for i in 0..5 {
    print(i)
}

// while 条件循环
mut n = 3
while n > 0 {
    print(n)
    n = n - 1
}

// match 模式匹配
description = match number {
    0 => "零",
    1 => "一",
    _ => "其他",
}
```

这些控制流结构都可以作为**表达式**使用——它们能计算出一个值。这是 YaoXiang 和许多传统语言的一个重要区别。

接下来的章节会逐一深入讲解每种控制流。建议按顺序阅读，因为它们之间存在自然的递进关系。
