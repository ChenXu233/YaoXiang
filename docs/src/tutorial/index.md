# 欢迎来到爻象宇宙！

> 「无极而太极。太极动而生阳，静而生阴。」
> —— 周敦颐《太极图说》

这里是爻象宇宙的中心。

在东方哲学中，「爻」是组成卦象的基本符号，阴阳变化，尽在其中。「象」是事物本质的外在表现，万象万物，皆有规律可循。

**爻象编程语言**，便是将这套古老的智慧融入现代类型系统的尝试。

## 什么是爻象？

想象你在搭建乐高积木：

| 概念 | 乐高 | 爻象 |
|------|------|------|
| 基本单元 | 积木块 | **类型** |
| 组合方式 | 拼接 | **函数映射** |
| 成品 | 飞机、汽车 | **程序** |

在爻象的世界里，**一切皆类型**：
- 数字是类型，文本是类型
- 类型本身也是类型（元类型）
- 函数是类型到类型的桥梁

## 核心特性

| 特性 | 爻象 | 其他语言举例 |
|------|------|-------------|
| 类型系统 | 依赖类型 + 参数化多态 | TypeScript |
| 内存管理 | 所有权模型（无 GC） | Rust |
| 异步编程 | 并作模型（无感并发） | async/await |
| 语法风格 | Python 风格 | Python |

**一句话总结**：像写 Python 一样流畅，像 Rust 一样安全。
