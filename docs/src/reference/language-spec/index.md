# 语言规范

瑶光语言的语法和语义规范文档。

## 目录

- [语法规范](./syntax.md) - 词法结构、语法规则、运算符优先级
- [类型系统](./type-system.md) - 基本类型、复合类型、泛型、trait
- [模块系统](./modules.md) - 模块定义、导入导出、作用域
- [并发模型](./concurrency.md) - 异步编程、并发原语、内存模型
- [标准库](./stdlib.md) - 核心库、IO库、数学库

## 概述

本规范定义了 YaoXiang 编程语言的核心语法和语义。它是语言的权威参考，面向编译器和工具实现者。

### 规范结构

1. **语法规范**：定义语言的词法结构、语法规则和运算符优先级
2. **类型系统**：定义语言的类型系统，包括基本类型、复合类型、泛型和 trait
3. **模块系统**：定义语言的模块系统，包括模块定义、导入导出和作用域
4. **并发模型**：定义语言的并发模型，包括异步编程、并发原语和内存模型
5. **标准库**：定义语言的标准库，包括核心库、IO库和数学库
