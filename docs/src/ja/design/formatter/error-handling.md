# エラーの処理

---

## §E1 フォーマットエラー

**§E1.1 構文エラー。** ソースコードに構文エラーが含まれている場合、フォーマッターは以下を行う必要がある：

1. 正しい部分を可能な限りフォーマットする
2. エラーのあるノードの元の内容を保持する
3. エラーの位置に `/* error */` プレースホルダーを挿入する

**§E1.2 設定エラー。** 設定ファイルの形式が正しくない場合は、明確なエラーメッセージを返す必要がある。

---

## §E2 終了コード

| 終了コード | 意味 |
|--------|------|
| 0 | 成功 |
| 1 | `--check` モードでフォーマットされていないファイルが見つかった |
| 2 | ファイルが見つからない、または設定エラー |
