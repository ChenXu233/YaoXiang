# コメント保持

---

## §C1 コメントの型

**§C1.1 単一行コメント。** `//`で始まり、保持されなければならない。

```
// これは単一行コメント
let x = 1;  // これは行末コメント
```

**§C1.2 複数行コメント。** `/* ... */`で囲み、ネストをサポート、保持されなければならない。

```
/* これは複数行コメント */
/* ネスト /* コメント */ */
```

**§C1.3 ドキュメンテーションコメント。** `///`で始まり、保持されなければならない。

```
/// これはドキュメンテーションコメント
fn foo() { ... }
```

---

## §C2 コメントの位置

**§C2.1 ファイルヘッダーコメント。** ファイル先頭のコメントは、ファイル先頭に保持されなければならない。

**§C2.2 文間コメント。** 文間のコメントは、元の位置に保持されなければならない。

**§C2.3 行末コメント。** 行末コメントは、同じ行の末尾に保持されなければならない。

---

## §C3 空行の保持

**§C3.1 空行の保持。** 元のコードの空行は、論理ブロックを区切るために保持されなければならない。
