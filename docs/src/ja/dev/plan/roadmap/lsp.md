> **モジュールステータス**：安定（5 項目の改善待ち）

## モジュールの概要

LSP モジュールは Language Server Protocol を実装し、エディタ/IDE にコードインテリジェンス機能を提供します。実装範囲は RFC-017 のステージ 0〜5 を網羅し、セマンティック tokens、リネーム、コードアクションなどの高度な機能を先行実装しています。

**コード量**：7,542 行（21 の Rust ファイル）
