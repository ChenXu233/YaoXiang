# 言語サーバー（LSP）

> **モジュールステータス**：完了（RFC 設計範囲外）
> **位置**：`src/lsp/`
> **最終更新**：2026-06-01

## モジュール概要

LSP モジュールは Language Server Protocol を実装し、エディタ/IDE にコードインテリジェンス機能を提供します。実装範囲は RFC-017 のステージ 0-5 をカバーし、セマンティック tokens、リネーム、コード操作などの高度な機能を先行実装しています。

**コード量**：7,542 行（21 の Rust ファイル）
