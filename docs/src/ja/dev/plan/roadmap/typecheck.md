## モジュール概要

タイプチェッカーは AST の意味解析と型推論を担当します。3 パス走査アーキテクチャを採用しています：型定義収集 → 関数シグネチャ収集 → 関数本体検査。完全な Hindley-Milner 型推論アルゴリズムを実装しています。

**コード量**：28,153 行（実装 15,383 行 + テスト 12,770 行）
