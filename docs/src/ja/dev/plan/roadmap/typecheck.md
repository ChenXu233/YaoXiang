# 型チェッカー（Typecheck）

> **モジュール状態**：完了
> **位置**：`src/frontend/core/typecheck/`
> **最終更新**：2026-06-01

## モジュール概要

型チェッカーは AST のセマンティック分析与び型推論を担当します。3パススキャンアーキテクチャを採用しています：型定義収集 → 関数シグネチャ収集 → 関数本体チェック。完全な Hindley-Milner 型推論アルゴリズムを実装しています。

**コード量**：28,153 行（実装 15,383 行 + テスト 12,770 行）
