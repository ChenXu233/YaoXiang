## モジュール概要

字句解析器は、ソースコード文字列を Token ストリームに変換する責務を持つ。古典的な文字ストリーム駆動アーキテクチャを採用しており、YaoXiang 言語の完全な字句仕様をサポートしている。

**コード量**：約 800 行（7 つのソースファイル）
