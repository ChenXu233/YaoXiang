# 字句解析器（Lexer）

> **モジュールステータス**：安定（改善点3件）
> **場所**：`src/frontend/core/lexer/`
> **最終更新**：2026-06-01

---

## モジュール概要

字句解析器はソースコード文字列をトークンストリームに変換する責務を負う。古典的な文字ストリーム駆動アーキテクチャを採用し、完全なYaoXiang言語の字句仕様をサポートする。

**コード量**：約800行（ソースファイル7個）
