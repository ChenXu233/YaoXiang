## モジュール概要

字句解析器はソースコード文字列をTokenストリームに変換する責任を持ちます。クラシックな文字ストリーム駆動アーキテクチャを採用しており、完全なYaoXiang言語字句仕様をサポートしています。

**コード量**：約800行（7つのソースファイル）
