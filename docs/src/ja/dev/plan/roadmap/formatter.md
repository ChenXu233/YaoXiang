# フォーマッタ（Formatter）

> **モジュール状態**：基本完成
> **位置**：`src/formatter/`
> **最終更新**：2026-06-01

---

## モジュールの概要

フォーマッタは、YaoXiang ソースコードを自動フォーマットする責務を負います。式、 文、 型などの完全な AST ノードフォーマットをサポートしています。

**コード量**：2,397 行（19 個のソースファイル）

---

## 機能一覧

### 実装済みの機能

**式のフォーマット**（handlers/expr.rs、全 Expr 変体を網羅）：
- ✅ リテラル：Int / Float / Bool / Char / String（エスケープシーケンスを含む）
- ✅ 変数参照
- ✅ 二項演算子 / 単項演算子（行幅超過時に自動改行）
- ✅ 関数呼び出し（名前付き引数を含む）
- ✅ 関数定義（式形式 fn def）
- ✅ if/elif/else
- ✅ match 式（pattern の整列、过长时换行）
- ✅ for / while ループ（ラベルを含む）
- ✅ コードブロック（コメント保持、行末コメント処理）
- ✅ return / break / continue
- ✅ 型変換（as）
- ✅ タプル / リスト / リスト内包表記 / 辞書
- ✅ インデックス / フィールドアクセス（チェーン呼び出しの改行）
- ✅ try 演算子（?）
- ✅ ref / borrow（& / &mut）
- ✅ unsafe ブロック
- ✅ eval ブロック（@block / @auto / @eager）
- ✅ spawn ブロック
- ✅ ラムダ式（単一式シンプル形式）
- ✅ f-string（フォーマット仕様を含む）
- ✅ Error ノード（`/* error */` プレースホルダを挿入）

**文のフォーマット**（handlers/stmt.rs）：
- ✅ 変数宣言（mut / 型注釈 / イニシャライザ）
- ✅ for ループ文
- ✅ 統一バインディング文（関数 / 型 / メソッドバインディング）
- ✅ use インポート文（items、alias を含む）
- ✅ if 文
- ✅ 外部バインディング文（External / Anonymous / DefaultExternal）

**型のフォーマット**（handlers/types.rs、全 Type 変体を網羅）：
- ✅ 基本型：Int(size) / Float(size) / Char / String / Bytes / Bool / Void
- ✅ 名前付き型 / 構造体 / 名前付き構造体
- ✅ Union / Enum / Variant
- ✅ タプル / 関数型 / Option / Result
- ✅ 泛型 / 関連型 / Sum 型
- ✅ リテラル型 / 参照型 / ポインタ型 / MetaType

**その他の機能**：
- ✅ 区切りリスト自動改行（handlers/delimited.rs）
- ✅ コメント保持（source_map.rs）
- ✅ インポートソート（rules/sort_imports.rs）
- ✅ CLI コマンド：check / write / stdout モード（command.rs）
- ✅ 設定オプション：line_width / indent_width / use_tabs / single_quote / sort_imports（options.rs）
