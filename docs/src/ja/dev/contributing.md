# 貢献ガイド

YaoXiang プロジェクトへのコードとドキュメントの貢献を歓迎します！

## 貢献方法

1. リポジトリを **Fork** する
2. フィーチャーブランチを作成：`git checkout -b feature/xxx`
3. 変更をコミット：`git commit -m "feat: xxx"`
4. ブランチをプッシュ：`git push origin feature/xxx`
5. **Pull Request** を作成

## コード規範

- プロジェクトのコードスタイルに準拠する
- テスト合格を確認する
- 関連するドキュメントを更新する

## コミット規範

```
feat: 新機能
fix: バグ修正
docs: ドキュメント更新
refactor: リファクタリング
test: テスト関連
chore: ビルド/ツール更新
```
