# 警告コード

このドキュメントは、YaoXiang コンパイラが生成する可能性のある警告コードの一覧です。警告はコンパイルを阻止しませんが、コードに潜在的な問題が存在する可能性を示しています。

## 設定

`yaoxiang.toml` で警告の動作を設定できます：

```toml
[lint]
# デッドコード警告のレベル：off | warn | deny
dead-code = "warn"
```

- `off`：警告を無効化する
- `warn`：警告を表示する（デフォルト）
- `deny`：警告をエラーとして扱い、コンパイルを阻止する

## 警告一覧

### W1001: 未使用の公開関数

**原因**：公開された関数が、どのコードからも呼び出されていません。

```yaoxiang
pub dead_function: () -> Void = { }  // W1001: 未使用の公開関数

main = {
    // dead_function は一度も呼び出されない
}
```

**推奨事項**：
- 関数を外部から使用する必要がない場合は、`pub` 修飾子を削除してください
- 関数を残す必要があるがまだ使用されていない場合は、設定で `dead-code = "off"` を指定できます

---

### W1002: 未使用の公開型

**原因**：公開された型（type エイリアスまたはカスタム型）が一度も使用されていません。

**例**：
```yaoxiang
DeadType: Type = Int  // W1002: 未使用の公開型

main = {
    x = 42
}
```

**推奨事項**：
- 型を公開する必要があるがまだ使用されていない場合は、この警告を無視してください

---

### W1003: 未使用のインポート

**原因**：`use` 文でインポートされたモジュールまたはシンボルが一度も使用されていません。

```yaoxiang
use std.json  // W1003: 未使用のインポート

main = {
    // json モジュールは一度も使用されない
}
```

**推奨事項**：
- 未使用のインポートを削除してコードを整理してください
- インポートを保持する必要がある場合（副作用のため）、`use std.json.*` の使用やコメントでの説明の追加を検討してください

---

### W1004: 未使用の公開変数

**原因**：`pub` で公開された変数が一度も読み取られていません。

**例**：
```yaoxiang
pub dead_var = 42  // W1004: 未使用の公開変数

main = {
    // dead_var は一度も読み取られない
}
```

**推奨事項**：
- 不要な `pub` 修飾子を削除してください
- 変数を公開する必要があるがまだ使用されていない場合は、この警告を無視してください

---

### W1005: 未使用の公開メソッド

**原因**：型上で公開されたメソッドが一度も呼び出されていません。

**例**：
```yaoxiang
Foo: Type = { value: Int }

pub Foo.dead_method: (self: Foo) -> Void = { }  // W1005: 未使用の公開メソッド

main = {
    foo = Foo(1)
    // dead_method は一度も呼び出されない
}
```

**推奨事項**：
- 不要な `pub` 修飾子を削除してください
- メソッドを残す必要があるがまだ使用されていない場合は、この警告を無視してください

---

## 警告レベルの詳細

| レベル | 効果 |
|------|------|
| `off` | この警告を完全に無効化する |
| `warn` | 警告を表示するがコンパイルを続行する（デフォルト） |
| `deny` | 警告をエラーとして扱い、コンパイルを阻止する |

### 使用シーン

- **開発中**：`warn` レベルを使用してコードの潜在的な問題を把握する
- **リリース前**：`deny` レベルを使用して未使用のコードがないことを確認する
- **レガシーコード**：`off` レベルを使用して一時的に警告を無視する

---

## エラーコードとの違い

警告コードは `W` プレフィックス（例：W1001）を使用し、エラーコードは `E` プレフィックス（例：E1001）を使用します。

- **エラー (Error)**：コンパイルを阻止し、必ず修正が必要
- **警告 (Warning)**：潜在的な問題を知らせるもので、修正は任意
