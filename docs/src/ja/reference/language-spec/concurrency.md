## 所有権モデル（要約）

完全なモデルについては [型システム規範](./type-system.md) 第11章、第12章を参照してください的安全性モデルの要約のみを示します：

```yaoxiang
// Move（デフォルト、ゼロコピー）
p2 = p

// ref（コンパイラが Rc または Arc を自動選択）
shared = ref p

// clone（明示的な深複製）
p2 = p.clone()

// unsafe（生ポインタ）
unsafe {
    ptr: *Point = &p
    (*ptr).x = 0.0
}
```

| セマンティクス | 説明 | コスト |
|------|------|------|
| Move | デフォルト，所有権移転 | ゼロ |
| `&T` / `&mut T` | 借用トークン、ゼロサイズのコンパイル時権限証明 | ゼロ |
| `ref` | コンパイラが Rc（単一タスク）/ Arc（跨タスク）を自動選択 | 需要に応じる |

## spawn 構文（抜粋）

完全な構文については [構文規範](./syntax.md) 3.10節を参照してください。

```yaoxiang
// spawn ブロック
(result_a, result_b) = spawn {
    parse(fetch("url1")),
    parse(fetch("url2"))
}

// spawn ループ
results = spawn for item in items {
    process(item)
}
```
