## 概要

本仕様書は YaoXiang プログラミング言語のコア構文と意味論を定義します。言語の権威ある参照文書であり、コンパイラおよびツール開発者向けです。

### 仕様書の構成

1. **構文仕様**：言語の字句構造、文法規則、演算子優先順位を定義します
2. **型システム**：言語の型システムを定義します。基本型、複合型、ジェネリクス、trait を含みます
3. **モジュールシステム**：言語のモジュールシステムを定義します。モジュール定義、インポート/エクスポート、スコープを含みます
4. **並行モデル**：言語の並行モデルを定義します。非同期プログラミング、並行プリミティブ、メモリモデルを含みます
5. **標準ライブラリ**：言語の標準ライブラリを定義します。コアライブラリ、IOライブラリ、数学ライブラリを含みます
