## 概要

本仕様書は YaoXiang プログラミング言語の中核的な構文と意味論を定義します。これは言語の権威ある参照文書であり、コンパイラとツールの実装者を対象としています。

### 仕様書の構造

1. **構文仕様**：言語の字句構造、構文規則、演算子の優先順位を定義します
2. **型システム**：言語の型システムを定義します。基本型、複合型、ジェネリクス、trait を含みます
3. **モジュールシステム**：言語のモジュールシステムを定義します。モジュール定義>import export、スコープを含みます
4. **並行モデル**：言語の並行モデルを定義します。非同期プログラミング、並行プリミティブ、メモリモデルを含みます
5. **標準ライブラリ**：言語の標準ライブラリを定義します。コアライブラリ、IOライブラリ、数学ライブラリを含みます
