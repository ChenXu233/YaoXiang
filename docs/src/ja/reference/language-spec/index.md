## 概要

本仕様は YaoXiang プログラミング言語の中核的な構文と意味論を定義する。コンパイラおよびツール実装者向けの言語の権威あるリファレンスである。

### 仕様構造

1. **構文仕様**：言語の字句構造、構文規則、演算子の優先順位を定義
2. **型システム**：基本型、複合型、泛型、trait を含む言語の型システムを定義
3. **モジュールシステム**：モジュールの定義、インポート/エクスポート、スコープを含む言語のモジュールシステムを定義
4. **並行モデル**：非同期プログラミング、並行プリミティブ、メモリモデルを含む言語の並行モデルを定義
5. **標準ライブラリ**：コアライブラリ、IOライブラリ、数学ライブラリを含む言語の標準ライブラリを定義
