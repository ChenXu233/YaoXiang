## 概要

本仕様は YaoXiang プログラミング言語の核となる構文と意味論を定義する。言語の权威的な参照文書であり、コンパイラおよびツールの実装者を対象とする。

### 仕様の構成

1. **構文仕様**：言語の字句構造、構文規則、演算子の優先順位を定義
2. **型システム**：言語の型システムを定義。基本型、複合型、泛型、trait を含む
3. **モジュールシステム**：言語のモジュールシステムを定義。モジュール定義、インポート/エクスポート、スコープを含む
4. **並行モデル**：言語の並行モデルを定義。非同期プログラミング、并发プリミティブ、メモリモデルを含む
5. **標準ライブラリ**：言語の標準ライブラリを定義。コアライブラリ、IOライブラリ、数学ライブラリを含む
