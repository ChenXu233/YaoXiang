# エラーコード

本文書はYaoXiangパッケージマネージャーが返す可能性のあるエラーとその処理方法を列出します。

## エラー一覧

### E0100: プロジェクトは既に存在します

```
Error: Project already exists: <path>
```

**原因**：作成を試みたプロジェクトディレクトリが既に存在します。

**対処方法**：
1. 別のプロジェクト名を選択する
2. 既存のディレクトリを削除または移動する

---

### E0101: 有効なプロジェクトではありません

```
Error: Not a YaoXiang project: yaoxiang.toml not found
```

**原因**：現在のディレクトリまたは指定されたディレクトリに`yaoxiang.toml`ファイルが含まれていません。

**対処方法**：
1. プロジェクトディレクトリ内でコマンドを実行していることを確認する
2. `yaoxiang init`を使用して新規プロジェクトを作成する

---

### E0102: 依存関係が存在しません

```
Error: Dependency not found: <name>
```

**原因**：存在しない依存関係を操作しようとしました。

**対処方法**：
1. 依存関係名の綴りが正しいか確認する
2. `yaoxiang list`を使用して既存の依存関係を確認する

---

### E0103: 依存関係は既に存在します

```
Error: Dependency already exists: <name>
```

**原因**：既に存在する依存関係を追加しようとしました。

**対処方法**：
1. バージョンの更新が必要な場合は、まず`yaoxiang rm`で削除する
2. または既存の依存関係を直接使用する

---

### E0104: 無効なmanifest形式

```
Error: Invalid yaoxiang.toml format: <details>
```

**原因**：`yaoxiang.toml`ファイルの形式が正しくありません。

**対処方法**：
1. TOML構文が正しいか確認する
2. すべての必須フィールドが存在することを確認する
3. 構文エラー（引用符やかっこの欠落など）がないか確認する

---

### E0105: IOエラー

```
Error: IO error: <details>
```

**原因**：ファイルの読み書き操作に失敗しました。

**よくある原因**：
- ディスク容量不足
- 権限不足
- ファイルが他のプログラムによって使用中

**対処方法**：
1. ディスク容量を確認する
2. ファイルの権限を確認する
3. ファイルを使用している可能性のある他のプログラムを閉じる

---

### E0106: TOML解析エラー

```
Error: TOML parse error: <details>
```

**原因**：TOMLファイルの形式エラー。

**対処方法**：
1. TOML構文を検証する
2. 特殊文字が正しくエスケープされているか確認する

---

## よくある質問

### Q: 依存関係のインストールに失敗した場合は？

1. ネットワーク接続を確認する
2. 依存関係名とバージョンが正しいか確認する
3. `yaoxiang update`を使用して更新を試みる

### Q: バージョンの競合が発生した場合は？

`yaoxiang.toml`に互換性のない依存関係バージョン要件があるかどうかを確認します。

### Q: vendorディレクトリが破損した場合は？

`vendor`ディレクトリを削除してから`yaoxiang install`を再度実行します。
