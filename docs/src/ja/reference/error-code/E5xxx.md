# E5xxx：モジュールとインポート

> `src/util/diagnostic/codes/` から自動生成

## エラーリスト

## E5001：モジュールが見つかりません

**カテゴリ**: Module

**メッセージ**: 参照されたモジュールが存在しません

**ヘルプ**: モジュールのパスを確認し、存在することを確認してください

---

## E5002：インポートエラー

**カテゴリ**: Module

**メッセージ**: モジュールのインポートに失敗しました

**ヘルプ**: インポートパスとモジュールの解決方法を確認してください

---

## E5003：エクスポートが見つかりません

**カテゴリ**: Module

**メッセージ**: モジュール内に参照されたエクスポートが存在しません

**ヘルプ**: モジュールのエクスポートを確認してください

---

## E5004：循環依存

**カテゴリ**: Module

**メッセージ**: モジュール間に循環依存があります

**ヘルプ**: 循環依存を削除するようにリファクタリングしてください
