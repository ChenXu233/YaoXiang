# E6xxx：ランタイムエラー

> `src/util/diagnostic/codes/` より自動生成

## エラー一覧

## E6001：ゼロ除算

**カテゴリー**: ランタイム

**メッセージ**: ゼロ除算を試みました

**ヘルプ**: ゼロ除算を防ぐためにチェックを追加してください

---

## E6002：null値逆参照

**カテゴリー**: ランタイム

**メッセージ**: null値にアクセスしようとしました

**ヘルプ**: 値にアクセスする前にnullチェックを追加してください

---

## E6003：配列インデックスが範囲外

**カテゴリー**: ランタイム

**メッセージ**: 配列インデックスがランタイム時に範囲外です

**ヘルプ**: インデックスが配列の範囲内であることを確認してください

---

## E6004：スタックオーバーフロー

**カテゴリー**: ランタイム

**メッセージ**: 再帰の深さがスタックの制限を超えました

**ヘルプ**: 再帰の深さを減らすか、反復を使用してください

---

## E6005：アサーション失敗

**カテゴリー**: ランタイム

**メッセージ**: ランタイム時にアサーションが失敗しました

**ヘルプ**: アサーションの条件修正または有効な入力の提供を行ってください
