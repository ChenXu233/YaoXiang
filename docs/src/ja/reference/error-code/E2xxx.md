# E2xxx：意味論的分析

> `src/util/diagnostic/codes/` から自動生成

## エラーリスト

## E2001：スコープエラー

**カテゴリ**: 意味論

**メッセージ**: 変数が現在のスコープに存在しません

**ヘルプ**: 変数のスコープを確認し、ここからアクセス可能であることを保証してください

---

## E2002：重複定義

**カテゴリ**: 意味論

**メッセージ**: 変数が同じスコープ内で複数回定義されています

**ヘルプ**: 重複した定義をリネームまたは削除してください

---

## E2003：所有権エラー

**カテゴリ**: 意味論

**メッセージ**: 所有権の制約が満たされていません

**ヘルプ**: 値の所有権のセマンティクスを確認してください

---

## E2010：不変代入

**カテゴリ**: 意味論

**メッセージ**: 不変変数を変更しようとしています

**ヘルプ**: 可変変数を宣言するには 'mut' を使用してください

---

## E2011：未初期化の使用

**カテゴリ**: 意味論

**メッセージ**: 未初期化変数の使用

**ヘルプ**: 使用前に変数を初期化してください

---

## E2012：可変性衝突

**カテゴリ**: 意味論

**メッセージ**: 不変コンテキストで可変参照を使用

**ヘルプ**: 参照の変更可能性は使用コンテキストと一致していることを確認してください
