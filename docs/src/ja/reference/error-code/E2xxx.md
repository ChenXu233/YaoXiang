# E2xxx：意味解析

> `src/util/diagnostic/codes/` から自動生成

## エラー一覧

## E2001：スコープエラー

**カテゴリ**: Semantic

**メッセージ**: 変数が現在のスコープに存在しません

**ヘルプ**: 変数のスコープを確認し、ここでアクセス可能であることを保証してください

---

## E2002：重複定義

**カテゴリ**: Semantic

**メッセージ**: 同じスコープ内で変数が複数回定義されています

**ヘルプ**: 名前を変更するか、重複した定義を削除してください

---

## E2003：所有権エラー

**カテゴリ**: Semantic

**メッセージ**: 所有権制約が満たされていません

**ヘルプ**: 値の所有権セマンティクスを確認してください

---

## E2010：不変代入

**カテゴリ**: Semantic

**メッセージ**: 不変変数を変更しようとしました

**ヘルプ**: `mut` を使用して可変変数を宣言してください

---

## E2011：未初期化の使用

**カテゴリ**: Semantic

**メッセージ**: 初期化されていない変数を使用しています

**ヘルプ**: 使用前に変数を初期化してください

---

## E2012：可変性衝突

**カテゴリ**: Semantic

**メッセージ**: 不変コンテキストで可変参照を使用しています

**ヘルプ**: 参照の可変性が使用コンテキストと一致していることを確認してください
