## エラー一覧

## E1001：未知の変数

**カテゴリ**: 型チェック

**メッセージ**: 参照された変数が定義されていません

**ヘルプ**: 変数名が正しく入力されているか確認するか、まず変数を定義してください

## E1002：型の不一致

**カテゴリ**: 型チェック

**メッセージ**: 期待される型が実際の型と一致しません

**ヘルプ**: 正しい型を使用するか、型変換を追加してください

---

**ヘルプ**: 型名が正しく入力されているか確認してください
