# 制御フロー

前章では関数の定義方法を学びました。続いて、コードに「意思決定」と「繰り返し実行」の能力を持たせる方法を学びましょう——これが**制御フロー**です。

YaoXiang は五つの制御フロー構造を提供しており、それぞれが固有の役割を担います。

| 制御フロー | 用途 | 一言説明 |
|--------|------|------------|
| `if-elif-else` | 条件分岐 | 条件に応じて実行パスを選択する |
| `for` | 繰り返しループ | コレクション内の各要素を順次処理する |
| `while` | 条件ループ | 条件が成立する限りループを続ける |
| `break` / `continue` | ループ制御 | ループを途中で抜ける、または現在の反復をスキップする |
| `match` | パターンマッチング | 値の構造に基づいて分岐処理する |

まずは簡単に見てみましょう。

```yaoxiang
# if は式であり、値を返すことができる
status = if score >= 60 { "及格" } else { "不及格" }

# for で範囲を反復する
for i in 0..5 {
    println(i)
}

# while で条件ループ
mut n = 3
while n > 0 {
    println(n)
    n = n - 1
}

# match でパターンマッチング
description = match number {
    0 => "零",
    1 => "一",
    _ => "其他",
}
```

これらの制御フロー構造は、いずれも**式**として使用できます——つまり、値を算出することができます。これは YaoXiang と多くの従来言語との重要な違いです。

続く章では、それぞれの制御フローを一つずつ詳しく解説します。自然な段階的関係があるため、順に読んでいくことをおすすめします。
