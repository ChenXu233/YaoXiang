# 制御フロー

前の章では関数を定義する方法を学びました。ここでは、コードに「意思決定」と「繰り返し実行」の能力を持たせる方法を学びます。それが**制御フロー**です。

YaoXiang は 5 種類の制御フロー構造を提供しており、それぞれが独自の役割を担っています。

| 制御フロー | 用途 | 一言での説明 |
|--------|------|------------|
| `if-elif-else` | 条件分岐 | 条件に応じて実行パスを選択する |
| `for` | 反復処理 | コレクション内の各要素を順番に処理する |
| `while` | 条件ループ | 条件が成立する限りループを続ける |
| `break` / `continue` | ループ制御 | ループを途中で抜けたり、現在の反復をスキップしたりする |
| `match` | パターンマッチング | 値の構造に応じて分岐処理する |

まずは簡単に見てみましょう。

```yaoxiang
// if は式であり、値を返すことができます
status = if score >= 60 { "合格" } else { "不合格" }

// for は範囲を反復処理します
for i in 0..5 {
    print(i)
}

// while 条件ループ
mut n = 3
while n > 0 {
    print(n)
    n = n - 1
}

// match パターンマッチング
description = match number {
    0 => "零",
    1 => "一",
    _ => "その他",
}
```

これらの制御フロー構造はすべて**式**として使うことができます。つまり、値を計算できるということです。これは YaoXiang と多くの従来型言語の重要な違いです。

次の章では、それぞれの制御フローを順番に詳しく解説します。相互に自然な段階的な関係があるため、順番に読むことをおすすめします。
