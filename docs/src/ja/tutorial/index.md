# YaoXiang 宇宙へようこそ！

> 「无极而太极。太极动而生阳，静而生阴。」
> —— 周敦颐『太极図説』

这里是 YaoXiang 宇宙的中心。

在东方哲学中、「爻」は卦を構成する基本記号であり、陰陽の変化は全てその中に宿る。「象」は事物の本質的外在表現であり、万象万物は全て法則に則っている。

**爻象プログラミング言語**は、この古代の叡智を現代的类型システムに融合する試みである。

## 什么是 YaoXiang？

想象你在搭建乐高积木：

| コンセプト | レゴ | YaoXiang |
|------|------|----------|
| 基本ユニット | ブロック | **型（タイプ）** |
| 組み合わせ方 | 组み合わせ | **関数マッピング** |
| 完成品 | 飛行機、車 | **プログラム** |

在 YaoXiang の世界里，**一切皆型（タイプ）**：
- 数字は型（タイプ）であり、テキストも型（タイプ）
- 型自体も型である（元型）
- 関数は型から型への架け橋

## コア機能

| 機能 | YaoXiang | 他の言語の例 |
|------|---------|-------------|
| 型システム | 依存型 + パラメータ多相性 | TypeScript |
| メモリ管理 | 所有権モデル（GCなし） | Rust |
| 非同期プログラミング | 并作モデル（透過的並行性） | async/await |
| 構文スタイル | Python スタイル | Python |

**一句话总结**：Pythonのように流畅，Rustのように安全。
