# YaoXiangの世界へようこそ！

> 「无极而太极。太极动而生阳，静而生阴。」
> —— 周敦颐『太极図説』

这里是 YaoXiang 世界の中央です。

東洋哲学において、「爻」は卦を構成する基本記号であり、陰陽の変化はすべてこの中に秘められています。「象」は事物の本質から外見への反映であり、あらゆる万象はすべて法則に従っています。

**爻象プログラミング言語**は、この古代の智慧を現代の型システムに溶け込ませた試みです。

## 爻象とは？

レゴブロックを組み立てるようなものを想像してください：

| コンセプト | レゴ | 爻象 |
|------|------|------|
| 基本ユニット | ブロック | **型** |
| 組み合わせ方 | 拼接 | **関数マッピング** |
| 完成品 | 飛行機、車 | **プログラム** |

爻象の世界では、**すべてが型**です：
- 数字は型、テキストも型
- 型自体がまた型である（メタ型）
- 関数は型から型への架け橋

## コア機能

| 機能 | 爻象 | 他の言語の例 |
|------|------|-------------|
| 型システム | 依存型 + パラメータ多相 | TypeScript |
| メモリ管理 | 所有権モデル（GC なし） | Rust |
| 非同期プログラミング | 並作モデル（透過的並行性） | async/await |
| 構文スタイル | Python 風 | Python |

**一句话まとめ**：Python のような流れるような書き心地で、Rust のような安全性を。

- ✅ プログラミング言語設計を探究したい探索者
