# 爻象ユニバースへようこそ！

> 「無極にして太極。太極動じて陽を生じ、静じて陰を生ず。」
> —— 周敦頤『太極図説』

ここは爻象ユニバースの中心です。

東洋哲学において、「爻」は卦象を構成する基本記号であり、陰陽の変化はすべてその中にあります。「象」は事物本质の外在的表現であり、万象万物にはすべて法則があります。

**爻象プログラミング言語**は、この古来の知恵を現代的な型システム（type system）に取り入れる試みです。

## 爻象とは？

レゴブロックを組み立てることを想像してください：

| 概念 | レゴ | 爻象 |
|------|------|------|
| 基本ユニット | ブロック | **型** |
| 組み合わせ方 | 連結 | **関数マッピング** |
| 完成品 | 飛行機、車 | **プログラム** |

爻象の世界では、**すべてが型**です：
- 数値は型であり、テキストは型です
- 型自体も型です（メタ型）
- 関数は型から型への橋渡しです

## 核心的な特徴

| 特徴 | 爻象 | 他の言語の例 |
|------|------|-------------|
| 型システム | 依存型 + パラメータ多相 | TypeScript |
| メモリ管理 | 所有権（ownership）モデル（GC なし） | Rust |
| 非同期プログラミング | spawn モデル（待ち時間のない並列処理） | async/await |
| 構文スタイル | Python スタイル | Python |

**一言でまとめると**：Python のように滑らかに、Rust のように安全に。
