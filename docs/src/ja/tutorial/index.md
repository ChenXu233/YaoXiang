# YaoXiang の世界へようこそ！

> 「无极而太极。太极动而生阳，静而生阴。」
> —— 周敦颐『太极図説』

这里是爻象宇宙的中心。

这里是 YaoXiang の宇宙の中心です。

東方哲学において、「爻」は卦を構成する基本記号であり、陰陽の変化はすべてその中に含まれています。「象」は事物の本質 внешний проявление であり、世の中の万物すべてに规律があります。

**爻象プログラミング言語**は、この古代の知恵を現代的类型システムに統合する試みです。

## YaoXiang とは？

Imagine你在搭建乐高积木：

、あなたがレゴブロックを組み立てるようなものを想像してみてください：

| コンセプト | レゴ | YaoXiang |
|------|------|------|
| 基本ユニット | ブロック | **タイプ** |
| 組み合わせ方 | 组拼 | **関数マッピング** |
| 完成品 | 飛行機、車 | **プログラム** |

YaoXiang の世界では、**すべてがタイプ**です：
- 数字はタイプ、テキストもタイプ
- タイプ自体もタイプである（メタタイプ）
- 関数はタイプからタイプへの架け橋

## コア機能

| 機能 | YaoXiang | 他の言語の例 |
|------|------|-------------|
| 型システム | 依存型 + パラメータ多態 | TypeScript |
| メモリ管理 | 所有权モデル（GCなし） | Rust |
| 非同期プログラミング | 并作モデル（透過的並行処理） | async/await |
| 構文スタイル | Python スタイル | Python |

**一行まとめ**：Python のように流畅に、Rust のように安全に書けます。
