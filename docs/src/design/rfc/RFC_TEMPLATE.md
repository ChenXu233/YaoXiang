```markdown
#### 讨论记录

#### 决议

[最终决定及理由]
```

#### 讨论管理规则

| 阶段 | 操作 |
|------|------|
| 讨论中 | 记录在附录，保持开放状态 |
| 已解决 | 更新决议，标记状态，**更新正文相关内容** |
| 放弃 | 标记放弃原因，保留记录供参考 |

> **重要**: 讨论确定后，必须更新正文相关内容！讨论记录可保留作为历史参考。
