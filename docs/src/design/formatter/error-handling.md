# 错误处理

---

## §E1 格式化错误

**§E1.1 语法错误。** 当源代码包含语法错误时，格式化器应该：

1. 尽可能格式化正确的部分
2. 保留错误节点的原始内容
3. 在错误位置插入 `/* error */` 占位符

**§E1.2 配置错误。** 当配置文件格式错误时，应该返回明确的错误信息。

---

## §E2 退出码

| 退出码 | 含义 |
|--------|------|
| 0 | 成功 |
| 1 | `--check` 模式发现未格式化的文件 |
| 2 | 文件未找到或配置错误 |
