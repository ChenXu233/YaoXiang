# 注释保留

---

## §C1 注释类型

**§C1.1 单行注释。** 使用 `//` 开头，必须被保留。

```
// 这是单行注释
let x = 1;  // 这是行末注释
```

**§C1.2 多行注释。** 使用 `/* ... */` 包围，支持嵌套，必须被保留。

```
/* 这是多行注释 */
/* 嵌套 /* 注释 */ */
```

**§C1.3 文档注释。** 使用 `///` 开头，必须被保留。

```
/// 这是文档注释
fn foo() { ... }
```

---

## §C2 注释位置

**§C2.1 文件头注释。** 文件开头的注释必须保留在文件开头。

**§C2.2 语句间注释。** 语句之间的注释必须保留在原位置。

**§C2.3 行末注释。** 行末注释必须保留在同一行的末尾。

---

## §C3 空行保留

**§C3.1 空行保留。** 原始代码中的空行应该被保留，用于分隔逻辑块。
